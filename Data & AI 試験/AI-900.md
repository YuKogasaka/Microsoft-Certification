## 試験について (意訳)
**※2023/08/04 に英語版が改定 (アップデート) される予定です。**
- Exam AI-900: Microsoft Azure AIFundamentals–Skills Measured

### 試験の対象者
Azure AI Fundamentals 認定資格を取得するためには、Exam AI-900: Microsoft Azure AI Fundamentals が必要となります。
(Exam AI-900: Microsoft Azure AI Fundamentals is required to earn the Azure AI Fundamentals certification.)

この試験では、機械学習 (ML) と人工知能 (AI) の概念、および関連する Microsoft Azure サービスに関する知識を評価します。
(This exam measures your knowledge of machine learning (ML) and artificial intelligence (AI) concepts and related Microsoft Azure services.)

試験の受験者は、技術的背景と非技術的背景の両方を持っている必要があります。データ サイエンスとソフトウェア エンジニアリングの経験は必要ありません。ただし、クラウドの基本とクライアントサーバー アプリケーションを理解していると有益です。
(As an exam candidate, you should have both technical and non-technical backgrounds. Data science and software engineering experience are not required; however, awareness of cloud basics and clientserver applications would be beneficial.)

### 対象となる領域 : 試験で評価されるスキル
試験は定期的にアップデートされ、役割を果たすために必要なスキルが反映されます。試験を受ける時期に応じて、スキル測定目標の 2 つのバージョンが含まれています。  
(Our exams are updated periodically to reflect skills that are required to perform a role. We have included two versions of the Skills Measured objectives depending on when you are taking the exam.)

常に、試験の英語版を最初にアップデートします。一部の試験は他の言語にローカライズされており、英語版がアップデートされてから約 8 週間後にアップデートされます。その他の利用可能な言語は、試験の詳細 Web ページの試験のスケジュールセクションに一覧表示されています。ご希望の言語で試験を利用できない場合は、試験を完了するために 30 分の追加時間をリクエストすることが可能です。  
(We always update the English language version of the exam first. Some exams are localized into other languages, and those are updated approximately eight weeks after the English version is updated. Other available languages are listed in the Schedule Exam section of the Exam Details webpage. If the exam isn't available in your preferred language, you can request an additional 30 minutes to complete the exam.)

注 : 評価される各スキル続いて記載している箇条書きの一覧は、スキルの評価項目 (どのように評価しているかを示す) となります。関連するトピックが試験内でカバーされている場合があります。  
(Note : The bullets that follow each of the skills measured are intended to illustrate how we are assessing that skill. Related topics may be covered in the exam.)

注 : ほとんどの問題は、GA (General Availability) となった機能に関するものです。プレビュー機能が一般的に使用されている場合、それらに関する問題が含まれる場合があります。  
(Note : Most questions cover features that are general availability (GA). The exam may contain questions on Preview features if those features are commonly used.)

### 機能グループ
#### 人工知能 (AI) ワークロードと考慮事項について説明する (Describe Artificial Intelligence workloads and considerations) (20—25%)
- 一般的な AI ワークロードの機能を特定する  
(Identify features of common AI workloads)
  - 異常検出ワークロードの機能を特定する  
  (identify features of anomaly detection workloads)
  - Computer Vision のワークロードを特定する  
  (identify computer vision workloads)
  - 自然言語処理のワークロードを特定する  
  (identify natural language processing workloads)
  - ナレッジ マイニングのワークロードを特定する  
  (identify knowledge mining workloads)
- 責任ある AI の指針となる原則を特定する  
(Identify guiding principles for responsible AI)
  - AI ソリューションの公平性に関する考慮事項を説明する  
  (describe considerations for fairness in an AI solution)
  - AI ソリューションの信頼性と安全性に関する考慮事項を説明する  
  (describe considerations for reliability and safety in an AI solution)
  - AI ソリューションのプライバシーとセキュリティに関する考慮事項を説明する  
  (describe considerations for privacy and security in an AI solution)
  - AI ソリューションにおける包括性に関する考慮事項を説明する  
  (describe considerations for inclusiveness in an AI solution)
  - AI ソリューションの透明性に関する考慮事項を説明する  
  (describe considerations for transparency in an AI solution)
  - AI ソリューションにおける説明責任に関する考慮事項を説明する  
  (describe considerations for accountability in an AI solution)
#### Azure での機械学習の基本原則について説明する (Describe fundamental principles of machine learning on Azure) (25—30%)
- 一般的な機械学習のタイプを特定する  
(Identify common machine learning types)
  - 回帰機械学習シナリオを特定する  
  (identify regression machine learning scenarios)
  - 分類機械学習シナリオを特定する  
  (identify classification machine learning scenarios)
  - クラスタリング機械学習シナリオを特定する  
  (identify clustering machine learning scenarios)
- コア機械学習の概念を説明する  
(Describe core machine learning concepts)
  - 機械学習用のデータセット内の機能とラベルを特定する  
  (identify features and labels in a dataset for machine learning)
  - 機械学習でトレーニングと検証のデータセットがどのように使用されるかを説明する  
  (describe how training and validation datasets are used in machine learning)
- Azure Machine Learning Studio のビジュアル ツールの機能を説明する  
(Describe capabilities of visual tools in Azure Machine Learning Studio)
  - ML UI を自動化する  
  (automated machine learning)
  - Azure Machine Learning デザイナーを使用する  
  (Azure Machine Learning designer)
#### Azure の Computer Vision ワークロードの機能について説明する (Describe features of computer vision workloads on Azure) (15—20%)
- Computer Vision ソリューションの一般的なタイプを特定する  
(Identify common types of computer vision solution)
  - 画像分類ソリューションの特徴を特定する  
  (identify features of image classification solutions)
  - オブジェクト検出ソリューションの機能を特定する  
  (identify features of object detection solutions)
  - 光学式文字認識ソリューションの機能を特定する  
  (identify features of optical character recognition solutions)
  - 顔検出、顔認識、および顔分析ソリューションの機能を特定する  
  (identify features of facial detection, facial recognition, and facial analysis solutions)
- Computer Vision タスク用の Azure ツールとサービスを特定する  
(Identify Azure tools and services for computer vision tasks)
  - Computer Vision サービスの機能を特定する  
  (identify capabilities of the Computer Vision service)
  - Custom Vision サービスの機能を特定する  
  (identify capabilities of the Custom Vision service)
  - Face サービスの機能を特定する  
  (identify capabilities of the Face service)
  - Form Recognizer サービスの機能を特定する  
  (identify capabilities of the Form Recognizer service)
#### Azure の自然言語処理 (NLP) ワークロードの機能について説明する (Describe features of Natural Language Processing (NLP) workloads on Azure) (25—30%)
- 一般的な NLP ワークロード シナリオの機能を特定する  
(Identify features of common NLP Workload Scenarios)
  - キーフレーズ抽出の機能と用途を特定する  
  (identify features and uses for key phrase extraction)
  - エンティティ認識の機能と用途を特定する  
  (identify features and uses for entity recognition)
  - 感情分析の機能と用途を特定する  
  (identify features and uses for sentiment analysis)
  - 言語モデリングの機能と用途を特定する  
  (identify features and uses for language modeling)
  - 音声認識と合成の機能と用途を特定する  
  (identify features and uses for speech recognition and synthesis)
  - 翻訳の機能と用途を特定する  
  (identify features and uses for translation)
- NLP ワークロード用の Azure ツールとサービスを特定する  
(Identify Azure tools and services for NLP workloads)
  - Language サービスの機能を特定する  
  (identify capabilities of the Language service)
  - Speech サービスの機能を特定する  
  (identify capabilities of the Speech service)
  - Translator サービスの機能を特定する  
  (identify capabilities of the Translator service)
- Azure での会話型 AI ソリューションに関する考慮事項を特定する

  - ボットの機能と用途を特定する
