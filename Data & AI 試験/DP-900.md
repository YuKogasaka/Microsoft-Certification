## 試験について (意訳)

- Exam DP-900: Microsoft Azure Data Fundamentals–Skills Measured

### 試験の対象者
この試験では、次のタスクを達成する能力を測定します : 主要なデータの概念を説明します。Azure のリレーショナル データに関する考慮事項を特定する。Azure で非リレーショナル データを操作する際の考慮事項について説明します。Azure での分析ワークロードについて説明します。
(This exam measures your ability to accomplish the following tasks: describe core data concepts; identify considerations for relational data on Azure; describe considerations for working with non-relational data on Azure; and describe an analytics workload on Azure.)

この試験の受験者は、リレーショナル データと非リレーショナル データの概念、およびトランザクションや分析などのさまざまな種類のデータ ワークロードに精通している必要があります。
(As an exam candidate, you should be familiar with the concepts of relational and non-relational data, and different types of data workloads such as transactional or analytical.)

(Our exams are updated periodically to reflect skills that are required to perform a role. We have included two versions of the Skills Measured objectives depending on when you are taking the exam.)

Other available languages are listed in the Schedule Exam section of the Exam Details webpage. If the exam isn't available in your preferred language, you can request an additional 30 minutes to complete the exam.)

(Note : Most questions cover features that are general availability (GA). The exam may contain questions on Preview features if those features are commonly used.)

### 機能グループ
#### コア データの概念を説明する (Describe core data concepts) (25–30%)
- データを表現する方法を説明する  
(Describe ways to represent data)
  - 構造化データの機能を説明する  
(Describe features of structured data)
  - 半構造化の特徴を説明する  
(Describe features of semi-structured)
  - 非構造化データの機能を説明する  
(Describe features of unstructured data)
- データ ストレージ オプションを特定する  
(Identify options for data storage)
  - データ ファイルの一般的な形式を説明する  
(Describe common formats for data files)
  - データベースの種類を説明する  
(Describe types of databases)
- 一般的なデータ ワークロードについて説明する  
(Describe common data workloads)
  - トランザクション ワークロードの機能を説明する  
(Describe features of transactional workloads)
  - 分析ワークロードの機能を説明する  
(Describe features of analytical workloads)
- データ ワークロードの役割と責任を特定する  
(Identify roles and responsibilities for data workloads)
  - データベース管理者の責任について説明する  
(Describe responsibilities for database administrators)
  - データ エンジニアの責任について説明する  
(Describe responsibilities for data engineers)
  - データ アナリストの責任について説明する  
(Describe responsibilities for data analysts)
#### Azure のリレーショナル データに関する考慮事項を特定する (Identify considerations for relational data on Azure) (20–25%)
- リレーショナルの概念を説明する  
(Describe relational concepts)
  - リレーショナル データの特徴を特定する  
(Identify features of relational data)
  - 正規化とそれが使用される理由を説明する  
(Describe normalization and why it is used)
  - 共通構造化クエリ言語 (SQL) ステートメントを特定する  
(Identify common structured query language (SQL) statements)
  - 共通データベース オブジェクトを特定する  
(Identify common database objects)
- リレーショナル Azure データ サービスを説明する  
(Describe relational Azure data services)
  - Azure SQL データベース、Azure SQL マネージド インスタンス、Azure 仮想マシン上の SQL Server を含む Azure SQL ファミリー製品について説明する  
(Describe the Azure SQL family of products including Azure SQL Database, Azure SQL Managed Instance, and SQL Server on Azure Virtual Machines)
  - オープンソース データベース システム用の Azure データベース サービスを特定する  
(Identify Azure database services for open-source database systems)
#### Azure で非リレーショナル データを操作する際の考慮事項を説明する (Describe considerations for working with non-relational data on Azure) (15–20%)
- Azure ストレージの機能を説明する  
(Describe capabilities of Azure storage)
  - Azure Blob ストレージを説明する  
(Describe Azure Blob storage)
  - Azure File ストレージを説明する  
(Describe Azure File storage)
  - Azure Table ストレージを説明する  
(Describe Azure Table storage)
- Azure Cosmos DB の機能を説明する  
(Describe capabilities and features of Azure Cosmos DB)
  - Azure Cosmos DB のユースケースを特定する  
(Identify use cases for Azure Cosmos DB)
  - Azure Cosmos DB API を説明する  
(Describe Azure Cosmos DB APIs)
#### Azure 上の分析ワークロードを説明する (Describe an analytics workload on Azure) (25–30%)
- 大規模分析の共通要素について説明する  
(Describe common elements of large-scale analytics)
  - データの取り込みと処理に関する考慮事項を説明する  
(Describe considerations for data ingestion and processing)
  - 分析データ ストアのオプションを説明する  
(Describe options for analytical data stores)
  - Azure Synapse Analytics、Azure Databricks、Azure HDInsight、Azure Data Factory を含むデータウェア ハウジング向けの Azure サービスについて説明する  
(Describe Azure services for data warehousing, including Azure Synapse Analytics, Azure Databricks, Azure HDInsight, and Azure Data Factory)
- リアルタイム データ分析の考慮事項を説明する  
(Describe consideration for real-time data analytics)
  - バッチ データとストリーミング データの違いを説明する  
(Describe the difference between batch and streaming data)
  - Azure Stream Analytics、Azure Synapse Data Explorer、Spark 構造化ストリーミングを含むリアルタイム分析のテクノロジーについて説明する  
(Describe technologies for real-time analytics including Azure Stream Analytics, Azure Synapse Data Explorer, and Spark structured streaming)
- Microsoft Power BI でのデータ視覚化について説明する  
(Describe data visualization in Microsoft Power BI)
  - Power BI の機能を特定する  
(Identify capabilities of Power BI)
  - Power BI のデータ モデルの機能を説明する  
(Describe features of data models in Power BI)
  - データの適切な視覚化を特定する  
(Identify appropriate visualizations for data)
