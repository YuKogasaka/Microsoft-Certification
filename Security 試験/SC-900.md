# SC-900 : Microsoft Security, Compliance, and Identity Fundamentals
Microsoft 認定資格「Microsoft Certified : Security, Compliance, and Identity Fundamentals」の対象試験です。

- Exam SC-900: Microsoft Security, Compliance, and Identity Fundamentals - Learn | Microsoft Docs  
https://docs.microsoft.com/learn/certifications/exams/sc-900

## 試験について (意訳)
**※2023/02/07 に英語版が改定 (アップデート) される予定です。**
- Study Guide - Exam SC-900: Microsoft Security, Compliance, and Identity Fundamentals  
https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE4Mr81

この認定資格は、クラウドベースおよび関連する Microsoft サービス全体のセキュリティ、コンプライアンス、ID (SCI) の基礎を理解しようとしている人を対象としています。  
(This certification is targeted to those looking to familiarize themselves with the fundamentals of security, compliance, and identity (SCI) across cloud-based and related Microsoft services.)

つまり、ビジネス関係者、新規/既存 IT プロフェッショナル、もしくは Microsoft のセキュリティ、コンプライアンス、および ID ソリューションに関心のある学生を含む方が対象者となります。  
(This is a broad audience that may include business stakeholders, new or existing IT professionals, or students who have an interest in Microsoft security, compliance, and identity solutions.)

さらに、Microsoft Azure と Microsoft 365 に精通している必要があり、Microsoft のセキュリティ、コンプライアンス、および ID ソリューションがこれらのソリューション領域にまたがって、全体的でエンド ツー エンドのソリューションを提供する方法を理解しておく必要があります。  
(Candidates should be familiar with Microsoft Azure and Microsoft 365 and want to understand how Microsoft security, compliance, and identity solutions can span across these solution areas to provide a holistic and end-to-end solution.)

Other available languages are listed in the Schedule Exam section of the Exam Details webpage. If the exam isn’t available in your preferred language, you can request an additional 30 minutes to complete the exam.)

### 機能グループ
#### セキュリティ、コンプライアンス、および ID の概念を説明する (Describe the concepts of security, compliance, and identity) (10-15%)
- セキュリティやコンプライアンスの概念を説明する  
  (Describe security and compliance concepts)
  - 共有責任モデルを説明する  
    (Describe the shared responsibility model)
  - 高度な防御 (Defense in Depth) を説明する  
    (Define defense in depth)
  - ゼロトラスト手法を説明する  
    (Describe the Zero-Trust model)
  - 暗号化とハッシュを説明する  
    (Describe encryption and hashing)
  - コンプライアンスの概念を説明する  
    (Describe compliance concepts)
- ID の概念を定義する  
  (Define identity concepts)
  - 主要なセキュリティ境界として ID を定義する  
    (Define identity as the primary security perimeter)
  - 認証を定義する  
    (Define authentication)
  - 承認を定義する  
    (Define authorization)
  - ID プロバイダーを説明する  
    (Describe identity providers)
  - Active Directory を説明する  
    (Describe Active Directory)
  - フェデレーションの概念を説明する  
    (Describe the concept of Federation)
#### Microsoft Entra の一部である Azure Active Directory (Azure AD) の機能を説明する (Describe the capabilities of Microsoft Azure Active Directory (Azure AD), part of Microsoft Entra) (25-30%)
- Azure AD の基本的な ID サービスと ID の種類を説明する  
  (Describe the basic identity services and identity types of Azure AD)
  - Azure AD を説明する  
    (Describe Azure AD)
  - Azure AD の ID を説明する  
    (Describe Azure AD identities)
  - ハイブリッド ID を説明する  
    (Describe hybrid identity)
  - 様々な外部 ID の種類を説明する  
    (Describe the different external identity types)
- Azure AD の認証機能を説明する  
  (Describe the authentication capabilities of Azure AD)
  - Azure AD で使用可能な認証方法を説明する
    (Describe the authentication methods available in Azure AD)
  - 多要素認証について説明する  
    (Describe Multi-factor Authentication)
  - セルフサービスのパスワード リセットについて説明する  
    (Describe self-service password reset)
  - Azure AD で利用可能なパスワード保護および管理機能について説明する  
    (Describe password protection and management capabilities available in Azure AD)
- Azure AD のアクセス管理機能を説明する  
  (Describe access management capabilities of Azure AD)
  - 条件付きアクセスを説明する  
    (Describe conditional access)
  - Azure AD ロールのメリットを説明する  
    (Describe the benefits of Azure AD roles)
  - Azure AD ロールベースのアクセス制御を説明する  
    (Describe the benefits of Aure AD role-based access control)
- Azure AD の ID 保護と ガバナンス機能を説明する  
  (Describe the identity protection and governance capabilities of Azure AD)
  - Azure AD の ID ガバナンスを説明する  
    (Describe identity governance in Azure AD)
  - エンタイトルメント管理とアクセス レビューを説明する  
    (Describe entitlement management and access reviews)
  - Azure AD Privileged Identity Managemen (PIM) の機能を説明する  
    (Describe the capabilities of Azure AD Privileged Identity Management (PIM))
  - Azure AD Identity Protection を説明する  
    (Describe Azure AD Identity Protection)

  - Microsoft Defender for Cloud を説明する

  - Microsoft Sentinel がどのように統合された脅威保護を提供するかを説明する

- Microsoft 365 Defender を用いた脅威保護を説明する

    (Describe Microsoft Defender for Office 365 (formerly Office 365 ATP))
  - describe Microsoft Defender for Endpoint (以前は、Microsoft Defender ATP) を説明する

  - describe Microsoft Defender for Cloud Apps を説明する

  - describe the Microsoft 365 Defender ポータルを説明する  
    (Describe the Microsoft 365 Defender portal)
