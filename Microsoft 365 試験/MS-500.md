### 試験の対象者
この試験の対象者は、Microsoft 365 およびハイブリッド環境のセキュリティおよびコンプライアンスソリューションを実装、管理、および監視します。Microsoft 365 セキュリティ管理者は、M365 エンタープライズ環境をプロアクティブに保護し、脅威に対応し、調査を実行、データ ガバナンスを実施します。 Microsoft 365 セキュリティ管理者は、Microsoft 365 エンタープライズ管理者、ビジネスの利害関係者、およびその他のワークロード管理者と協力して、セキュリティ戦略を計画および実装し、ソリューションが組織のポリシーと規制に準拠していることを確認する必要があります。

また M365 ワークロードに精通しており、ID の保護、情報の保護、脅威からの保護、セキュリティ管理、およびデータ ガバナンスに関する強力なスキルと経験を持っている必要があります。この役割は M365 環境に焦点を当てており、ハイブリッド環境が含まれます。

### スキルの評価
注：評価される各スキルの下に記載しているリストは、スキルの評価項目です。このリストは、決定的でも網羅的でもありません。

注：試験内容について、ほとんどの場合、プレビュー機能はカバーされず、一部の機能は GA (General Availability) となった際に試験に追加されます。

#### ID とアクセスの実装と管理 (30-35%)
- Microsoft 365 ハイブリッド環境をセキュア化する
  - Azure AD 認証オプションを計画する
  - Azure AD 同期オプションを計画する
  - Azure AD Connect イベントを監視、トラブルシュートする
- ID をセキュア化する
  - Azure AD グループ メンバーシップを実装する
  - パスワード管理を実装する
  - ID ガバナンスを構成、管理する
- 認証の方法を実装する
  - サインオン セキュリティを計画する
  - 条件付きアクセス ポリシーを用いた多要素認証 (MFA) を実装する
  - MFA を管理、監視する
  - Windows Hello のようなデバイス認証方法を計画、実装する
  - Azure AD ユーザー認証オプションとセルフサービス パスワード管理を構成、管理する
- 条件付きアクセスを実装する
  - コンプライアンスと条件付きアクセス ポリシーを計画する
  - エンドポイント セキュリティに対するデバイスのコンプライアンスを構成、管理する
  - 条件付きアクセスを実装、管理する
- ロールベースのアクセス制御 (RBAC) を実装する
  - ロールを計画する
  - ロールを構成する
  - ロールを監査する
- Azure AD Privileged Identity Management (PIM) を実装する
  - Azure PIM を計画する
  - 資格の割り当てや管理者の役割をアクティブ化する
  - Azure PIM ロールの要求や割り当てを管理する
  - PIM の履歴とアラートを監視する
- Azure AD Identity Protection を実装する
  - ユーザー リスク ポリシーを実装する
  - サインイン リスク ポリシーを実装する
  - アラートを構成する
  - リスク イベントをレビュー、対応する
#### Threat Protection の実装と管理 (20-25%)
- エンタープライズ ハイブリッド脅威保護ソリューションを実装する
  - Microsoft Defender for Identity ソリューションを計画する
  - Microsoft Defender for Identity をインストール、構成する
  - Microsoft Defender for Identity を監視、管理する
- Implement device threat protectionデバイス脅威保護を実装する
  - Microsoft Defender for Endpoint ソリューションを計画する
  - Microsoft Defender for Endpoint を実装する
  - Microsoft Defender For Endpoint を管理、監視する
- デバイスとアプリケーションの保護を実装、管理する
  - デバイスとアプリケーションの保護を計画する
  - Microsoft Defender Application Guard を構成、管理する
  - Microsoft Defender Application Control を構成、管理する
  - Exploit Protection を構成、管理する
  - Secure Boot を構成する
  - Windows デバイス暗号化を構成、管理する
  - 非 Windows デバイス暗号化を構成、管理する
  - デバイス上のセキュア化するアプリケーション データを計画する
  - アプリケーション保護ポリシーを実装する
- Microsoft Defender for Office 365 を実装、管理する
  - Microsoft Defender for Office 365 を構成する
  - Microsoft Defender for Office 365 を監視する
  - Attack Simulator を用いたシミュレートされた攻撃を実行する
- Azure Sentinel を用いた Microsoft 365 Security を監視する
  - Azure Sentinel を計画、実装する
  - Azure Sentinel のプレイブックを構成する
  - Azure Sentinel を管理、監視する
  - Azure Sentinel の脅威に対応する
#### Information Protection の実装と管理 (15-20%)
- Office 365 へのデータ アクセスをセキュア化する
  - Customer Lockbox を実装、管理する
  - Office 365 コラボレーション ワークロードのデータ アクセスを構成する
  - 外部ユーザーに対する B2B 共有を構成する
- 機密ラベルを管理する
  - 機密ラベル ソリューションを計画する
  - 機密ラベルとポリシーを構成する
  - ラベル分析を構成、利用する
  - Teams、Sharepoint、OneDrive や Office アプリで機密ラベルを利用する
- データ損失防止 (DLP) を管理する
  - DLP ソリューションを計画する
  - DLP ポリシーを作成、管理する
  - 機密情報のタイプを作成、管理する
  - DLP レポートを監視する
  - manage DLP の通知を管理する
- Microsoft Cloud App Security を実装、管理する
  - Cloud App Security の実装を計画する
  - Microsoft Cloud App Security を構成する
  - Cloud App Discovery を管理する
  - Cloud アプリ カタログのエントリを管理する
  - Cloud App Security のアプリを管理する
  - Microsoft Cloud App Security を管理する
  - Cloud App Security コネクタと Oauth アプリを構成する
  - Cloud App Security ポリシーやテンプレートを構成する
  - Cloud App Security のアラート、レポート、ダッシュボードやログをレビュー、解釈、対応する
#### Microsoft 365 のガバナンスとコンプライアンス機能の管理 (25-30%)
- セキュリティ レポートを構成、分析する
  - Microsoft Endpoint Manager Admin Center を用いてデバイスセキュリティ状態を監視、管理する
  - Microsoft 365 Security Center を用いてセキュリティとダッシュボードを管理、監視する
  - Graph Security API を利用したカスタム セキュリティ レポートを計画する
  - アクションや推奨事項をレビューするためにセキュリティ スコア ダッシュボードを利用する
  - Microsoft 365 セキュリティ センターのアラート ポリシーを構成する
- 監査ログとレポートを管理、分析する
  - 監査とレポートを計画する
  - 監査ログ検索を実行する
  - コンプライアンス レポートやダッシュボードをレビュー、解釈する
  - 監査アラート ポリシーを構成する
- データ ガバナンスと保持を管理する
  - データ ガバナンスと保持を計画する
  - データ ガバナンス レポートとダッシュボードをレビュー、解釈する
  - 保持ラベルやポリシーを構成する
  - データ ガバナンス イベントのタイプを定義する
  - コミュニケーション コンプライアンス ポリシーを定義、管理する
  - 情報の保持を構成する
  - 削除された Office 365 データを検索、回復する
  - データ あー会bを構成する
  - 非アクティブなメールボックスを管理する
- 検索と調査を管理する
  - コンテンツ検索と eDiscovery を計画する
  - 検索と検出ツールを用いてアクセス権限を委任する
  - コンテンツ検索を実行するために検索と調査ツールを利用する
  - コンテンツ検索の結果をエクスポートする
  - eDiscovery ケースを管理する
- データ プライバシー規制コンプライアンスを管理する
  - Microsoft 365 の規制コンプライアンスを計画する
  - GDPR ダッシュボードとレポートをレビュー、管理する
  - データ主体の要求 (DSR) を管理する
  - Compliance Manager を管理する
  - Compliance Manager レポートをレビューする
  - Compliance Manager アセスメントとアクション アイテムを作成、実行する
