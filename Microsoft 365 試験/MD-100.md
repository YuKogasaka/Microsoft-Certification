## 評価されるスキルについて (意訳)
**※2022/11/04 に英語版が改定 (アップデート) される予定です。**
- Study Guide - Exam MD-100: Windows Client
https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE3VJVC

この試験の受験者は、ドメイン インフラストラクチャ内での Microsoft Windows クライアントのインストール、構成、および保守に関する専門知識を持つ IT プロフェッショナルです。責任として、Windows デバイスの信頼性とデバイスに保存されているデータの保護と維持が含まれます。また、Microsoft Entra 環境の一部である Microsoft Azure Active Directory (Azure AD) で Windows クライアントを管理します。さらに、Microsoft 365 エンタープライズ管理者、Microsoft 365 セキュリティ管理者、および Windows Server 管理者と協力して、組織のビジネス ニーズを満たす最新のデスクトップおよびデバイス戦略を実装することができます。  
(Candidates for this exam are IT professionals who have subject matter expertise in installing, configuring, and maintaining Microsoft Windows clients within a domain infrastructure. Responsibilities include protecting and maintaining the reliability of the Windows device and the data stored on the device. Candidates for this exam administer Windows clients in a Microsoft Azure Active Directory (Azure AD), part of Microsoft Entra environment. Candidates for this exam may collaborate with the Microsoft 365 enterprise administrator, the Microsoft 365 security administrator, and Windows Server administrators to implement a modern desktop and device strategy that meets the business needs of an organization.)

加えて、デバイス管理、クライアント コンピューター ネットワーク、ポリシーベースの管理、およびクラウドベースのソリューションに精通している必要があります。  
(Candidates for this exam must be familiar with device management, client computer networking, policybased management, and cloud-based solutions.)

### 機能グループ
#### Windows のインストールと構成 (Install and configure Windows) (20–25%)
- Windows クライアントをインストールする  
(Install Windows client)
  - 適切な Windows エディションを選択する  
  (Select the appropriate Windows edition)
  - インストールするハードウェアを準備する  
  (Prepare hardware for installation)
  - 手動クリーン インストールを実行する  
  (Perform a manual clean installation)
  - 以前のバージョンの Windows からアップグレードを計画、実装する  
  (Plan and implement an upgrade from a previous version of Windows)
  - Windows ADK を用いて Windows クライアント インストールをカスタマイズする  
  (Customize a Windows client installation by using the Windows ADK)
  - アクティベーションを構成、アクティベーションの問題をトラブルシュートする  
  (Configure activation and troubleshoot activation issues)
- Windows クライアントで Hyper-V を管理、利用する  
(Manage and use Hyper-V on Windows client)
  - Hyper-V を用いて仮想マシンを作成、構成する  
  (Create and configure virtual machines by using Hyper-V)
  - 仮想ハード ドライブを管理する  
  (Manage virtual hard drives)
  - 仮想ネットワークを管理する  
  (Manage virtual networks)
  - Hyper-V 設定を構成する  
  (Configure Hyper-V settings)
  - チェックポイントを構成、管理する  
  (Configure and manage checkpoints)
  - Windows Sandbox を有効化、利用する  
  (Enable and use Windows Sandbox)
- Windows 設定を構成する  
(Configure Windows settings)
  - システム設定を構成する  
  (Configure system settings)
  - Windows 10 および Windows 11 のユーザー インターフェイスを管理する  
  (Manage user interface in Windows 10 and Windows 11)
  - configure Microsoft Edge を構成する  
  (Configure Microsoft Edge)
  - 言語と地域を構成する  
  (Configure language and region)
  - プリンターやほかのデバイスとの接続を構成、トラブルシュートする  
  (Configure and troubleshoot connections to printers and other devices)
  - プロビジョニング パッケージを用いて Windows クライクライアントを構成する  
  (Configure Windows client by using provisioning packages)
  - スタートアップ オプションを構成する  
  (Configure startup options)
  - サービスを構成、管理する  
  (Configure and manage services)
  - オプションの機能をインストール、構成する  
  (Install and configure optional features)
#### 接続および記憶域の構成と管理 (Configure and manage connectivity and storage) (15–20%)
- ネットワークとアクセスを構成する  
(Configure networking and access)
  - クライアント IP 設定を構成する  
  (Configure client IP settings)
  - モバイル ネットワークを構成する  
  (Configure mobile networking
  - ビルトインのツールや Connection Manager Administration Kit (CMAK) を用いて VPN クライアントを構成する  
  (Configure VPN client by using built-in tools or Connection Manager Administration Kit (CMAK))
  - クライアント デバイスで証明書を構成、管理する  
  (Configure and manage certificates on client devices)
  - クライアントとの接続をトラブルシュートする  
  (Troubleshoot client connectivity)
- 記憶域を構成、管理する  
(Configure and manage storage)
  - ローカル記憶域を構成する  
  (Configure local storage)
  - Windows クライアントに OneDrive を構成する  
  (Configure OneDrive on Windows client)
  - ディスク クリーンアップや Storage Sense を用いてローカル ドライブを最適化する  
  (Optimize local drives by using Disk cleanup or Storage Sense)
  - ファイルやフォルダーのアクセス権限を構成する  
  (Configure file and folder permissions)
#### Windows の管理 (Maintain Windows) (30–35%)
- システムとデータの回復を実行する  
(Perform system and data recovery)
  - ブート処理、スタートアップ処理をトラブルシュートする  
  (Troubleshoot boot and startup processes)
  - Windows クライアントを回復する  
  (Recover Windows client)
  - ファイルを回復する  
  (Recover files)
  - 復元ポイントを作成、管理する  
  (Create and manage restore points)
  - 復元ポイントからリストアする  
  (Restore from restore points)
- Windows Updates を管理する  
(Manage Windows updates)
  - アップデートを構成する  
  (Configure updates)
  - Windows 配信の最適化を構成する  
  (Configure Windows delivery optimization)
  - グループ ポリシー設定を用いてアップデートを制御する  
  (Control updates by using group policy settings)
  - Windows Update for Business を用いてアップデートを構成する  
  (Configure updates by using Windows Update for Business)
  - アップデートをトラブルシュートする  
  (Troubleshoot updates)
- リモート管理を構成する  
(Configure remote management)
  - リモート デスクトップを構成する  
  (Configure Remote Desktop)
  - Windows Admin Center を構成する  
  (Configure Windows Admin Center)
  - PowerShell リモーティングおよび Windows リモート管理を構成する  
  (Configure PowerShell remoting and Windows Remote Management)
  - リモート アシストやクイック アシストを含むリモート アシスタント ツールを構成する  
  (Configure remote assistance tools including Remote Assist and Quick Assist)
- Windows を監視、管理する  
(Monitor and manage Windows)
  - イベント ログを構成、分析する  
  (Configure and analyze event logs)
  - パフォーマンスと信頼性を監視、管理する  
  (Monitor and manage performance and reliability)
  - スケジュールされたタスクを構成する  
  (Configure scheduled tasks)
  - レジストリを管理する  
  (Manage registry)
#### デバイスとデータの保護 (Protect devices and data) (25–30%)
- ユーザー、グループ、コンピューター オブジェクトを管理する  
(Manage users, groups, and computer objects)
  - ローカル ユーザーを管理する  
  (Manage local users)
  - ローカル ユーザー プロファイルを管理する  
  (Manage local user profiles)
  - ローカル グループを管理する  
  (Manage local groups)
  - Windows クライアントの Microsoft アカウントを管理する  
  (Manage Microsoft accounts on Windows client)
  - Windows クライアントにアクセスするため、Active Directory からユーザーとグループを有効にする  
  (Enable users and groups from Active Directory to access Windows client)
  - join computers to Active Directory  
  (Join computers to Active Directory)
  - サインイン オプションを構成する  
  (configure sign-in options)
  - 資格情報マネージャを用いて資格情報を管理する  
  (Manage credentials by using Credential Manager)
  - ユーザー アクセス制御 (UAC) を構成する  
  (Configure user account control (UAC))
  - ローカル管理者パスワード設定 (LAPS) を実装、管理する  
  (Implement and manage Local Administrator Password Solutions (LAPS))
- ローカルおよびグループ ポリシーを構成、管理する  
(Configure and manage local and group policies)
  - Windows クライアントのローカル ポリシーおよびドメイン グループ ポリシーをトラブルシュートする  
  (Troubleshoot local policies and domain group policies on Windows client)
  - セキュリティ ポリシー、ユーザー権限の割り当て、監査ポリシーを含む、ローカルおよびグループ ポリシーを構成、管理する  
  (Configure and manage local and group policies, including security policy, user rights assignment, and audit policy)
  - グループ ポリシーを用いて Windows クライアント設定を構成する  
  (Configure Windows client settings by using group policy)
- Windows クライアントのセキュリティ設定を管理する  
(Manage security settings on Windows client)
  - Bitlocker を実装する  
  (Implement BitLocker)
  - Windows クライアント ファイアウォールを構成、管理する  
  (Configure and manage Windows client firewall)
  - ウイルスと脅威の保護を管理する  
  (Manage virus and threat protection)
  - アプリケーションおよびブラウザー制御設定を管理する  
  (Manage application and browser control settings)
