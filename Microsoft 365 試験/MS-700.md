### 試験の対象者
この試験の対象者は、Microsoft 365 環境での効率的かつ効果的なコラボレーションやコミュニケーションを促進するため、 Microsoft Teams を管理する Microsoft Teams 管理者です。

また、Teams チャット、アプリ、チャネル、会議、音声/ビデオ会議、ライブ イベント、通話、および Microsoft Teams 認定デバイスを計画、展開、構成、および保護できる必要があります。加えて、icrosoft Teams を SharePoint、OneDrive、Exchange、Microsoft 365 Groups、Power Platform などの他のワークロード、および他の Microsoft アプリ、サードパーティ アプリ、カスタム アプリと統合した経験が必要となります。

Microsoft Teams 管理者は、ネットワーキング、テレフォニー、ライセンス、セキュリティ、コンプライアンス、ユーザーの採用、アプリ開発、サポートなどの他の機能とも連携します。

### スキルの評価
注：評価される各スキルの下に記載しているリストは、スキルの評価項目です。このリストは、決定的でも網羅的でもありません。

注：試験内容について、ほとんどの場合、プレビュー機能はカバーされず、一部の機能は GA (General Availability) となった際に試験に追加されます。

#### Microsoft Teams 環境の計画と構成 (35-40%)
- Microsoft Teams におけるネットワーク設定を計画、構成する
  - Microsoft Teams 音声、ビデオ、ミーティングやライブ イベントに対するネットワーク帯域幅容量を計算する
  - Network Planner を用いてネットワーク使用量を分析する
  - Microsoft Teams のアウトバウンド ネットワーク ポートとプロトコルの要件を指定する
  - Microsoft Teams の WAN オプティマイザー、プロキシ、およびロード バランサーの推奨事項を特定する
  - Microsoft Call Quality Dashboard の Tenant Data Upload を構成する
  - Microsoft Teams のレポーティング ラベルを構成する
  - Microsoft 365 ネットワーク接続テストとダッシュボードを用いてネットワークの準備状況を評価する
  - QoS ポート範囲と DSCP マーキングを構成する
- Microsoft Teams のライセンス要件を特定する
  - コンプライアンス機能が必要なライセンスを特定する
  - チームの高度なライフサイクル管理に適切なライセンスを特定する
  - ゲスト アクセスに適切なライセンスを到底する
  - 通話、Phone System、リソース アカウントに適切なライセンスを特定する
  - Microsoft Teams Rooms および Common Area Phones に適切なライセンスを特定する
  - Microsoft Teams のアプリに適切なライセンスを特定する
- Microsoft Teams のセキュリティとコンプライアンスの設定を計画、構成する
  - Microsoft Teams のアラートを計画する
  - 適切な管理役割を選択する
  - 脅威ポリシーを計画する
  - 保持ポリシーを計画、構成する
  - 機密ラベルを計画、構成する
  - データ損失防止 (DLP) ポリシーを計画する
  - Microsoft Teams の条件付きアクセスおよび MFA を計画する
  - 情報バリア ポリシーを計画する
- Microsoft Teams のガバナンスやライフサイクル管理を計画、実装する
  - Microsoft Teams アップデート ポリシーを用いて Microsoft Teams プレビュー機能を計画、管理する 
  - Microsoft Teams のポリシー パッケージを作成、管理する
  - ユーザーやグループへのポリシー割り当てを計画する
  - Microsoft 365 Groups 作成のポリシーをセットアップする
  - Microsoft 365 Groups の期限切れポリシーを構成する
  - Microsoft 365 Groups の名前付けポリシーを構成する
  - 1 つ以上のチームをアーカイブ、削除、アーカイブ解除する
  - 1 つ以上の Microsoft 365 Groups の削除を復元、トラブルシューティングする
  - メンバーとゲストの AzureAD アクセス レビューを管理する
  - PowerShell を用いて一括ユーザー操作を実行する
- 外部とゲスト ユーザーを構成、管理する
  - SharePoint と OneDrive 組織レベルの共有設定を構成する
  - OneDrive ユーザーの外部ファイル共有設定を管理する
  - SharePoint サイトの外部ファイル共有設定を管理する
  - 外部のアクセス フェデレーション ドメインを管理する
  - Microsoft Teams 管理センターから Microsoft Teams のゲスト ユーザーを管理する
  - Azure AD 管理センターから Microsoft Teams にゲスト アクセスを構成する
  - 追加するゲストに所有者を許可/拒否する
  - 特定のチームにゲスト アクセスを構成する
  - ゲストを削除する
- Microsoft Teams デバイスの構成と管理
  - Microsoft Teams for VDI を構成する
  - Teams デバイスに対して構成プロファイルを管理する
  - Microsoft Teams デバイスタグを管理する
  - デバイス設定やファームウェアを管理する
  - 新しいデバイスにリモート サインインをプロビジョニング、構成する
#### チャット、チーム、チャネル、アプリの管理 (25-30%)
- チームを作成、管理する
  - チームを作成する
  - チームのプライバシー レベルを管理する
  - 既定、もしくはカスタム テンプレートからチームを構成する
  - 既存のリソースからチームを作成する
  - 組織全体のチームを計画、管理する
  - チームにユーザーを追加、削除する
  - チームのユーザーの役割を割り当て、修正する
  - 動的メンバーシップを構成する
- チャネルを計画、管理する
  - チャネル タイプを計画する
  - チャネルを作成する
  - Microsoft Teams のチャネル設定を管理する
  - プライベート チャネル用の Microsoft Teams ポリシーを作成、管理する
  - プライベート チャネル メンバーシップを管理する
- チャットとコラボレーション エクスペリエンスを管理する
  - メッセージング ポリシーを作成、管理する
  - Microsoft Teams 設定から E メール統合を構成する
  - クラウド ファイル ストレージ オプションを管理する
  - 適切な共存モードを選択する
- Microsoft Teams のアプリを管理する
  - 組織全体のアプリ設定を管理する
  - アプリのアクセス権限ポリシーを作成、構成する
  - アプリのセットアップ ポリシーを作成、構成する
  - team チームにアプリ、コネクター、タブ、メッセージング拡張機能を追加する
  - アプリ ストアをカスタマイズする
  - Microsoft Teams にカスタム アプリを公開、構成する
#### 通話とミーティングの管理 (15-20%)
- ミーティング エクスペリエンスを管理する
  - ミーティング設定を構成する
  - ミーティング ポリシーを作成、構成する
  - 会議ブリッジを作成、管理する
  - ライブ イベント設定を構成する
  - ライブ イベント ポリシーを作成、管理する
- 電話番号を管理する
  - Calling Plan や Direct Routing をどのタイミングで使用するかを評価する
  - 緊急呼び出しポリシーを計画、構成する
  - 組織の緊急アドレスを追加、変更、削除する
  - ユーザー、サービス、および会議ブリッジの組織の電話番号をプロビジョニング、管理する
  - ユーザーの電話番号を割り当て、変更、削除する
  - リソース アカウントに電話番号を割り当てる
- Phone System for Microsoft Teams を管理する
  - コール パーク ポリシーを作成、管理する
  - 通話ポリシーを作成、管理する
  - 発信者 ID ポリシーを作成、管理する
  - ユーザー音声設定を管理する
  - ダイアル プランを作成、管理する
  - ダイヤル プランを解釈する
  - Health Dashboard for Direct Routing を用いて潜在的な問題を特定する
  - リソース アカウントを作成、管理する
  - 呼び出しキューを作成、管理する
  - 自動応答を作成、管理する
#### Microsoft Teams 環境の監視とトラブルシュート (10-15%)
- Microsoft Teams 環境を監視、レポートする
  - 音声およびミーティングの品質を管理、レポートする
  - 生産性スコア、チームの使用状況、アプリの使用状況、アクティブ ユーザー、会議ごとの指標を含むユーザーの採用に関してレポートする
  - チームの作成や削除を監視、レポートする
  - ゲスト アクセスを監視、レポートする
  - Microsoft Teams 管理センターから通知とアラート ルールを管理する
- 音声、ビデオ、クライアントの問題をトラブルシュートする
  - チャットおよびプレゼンスの問題をトラブルシュートする
  - クライアント ログを特定、収集、解釈する
  - Call Analytics を用いて問題をトラブルシュートする
  - Azure AD サインイン ログを用いて Microsoft Teams のサインイン問題をトラブルシュートする
  - Call Quality Dashboard を用いて潜在的な問題を特定する
