### 試験の対象者
この試験の対象者は、Microsoft 365 環境での効率的かつ効果的なコラボレーションやコミュニケーションを促進するため、 Microsoft Teams を管理する Microsoft Teams 管理者です。  
(Candidates for this exam are Microsoft Teams administrators who manage Microsoft Teams to facilitate efficient and effective collaboration and communication in a Microsoft 365 environment.)

また、Teams チャット、アプリ、チャネル、会議、音声/ビデオ会議、ライブ イベント、通話、および Microsoft Teams 認定デバイスを計画、デプロイ、構成、および保護できる必要があります。加えて、icrosoft Teams を SharePoint、OneDrive、Exchange、Microsoft 365 Groups、Power Platform を含む他のワークロード、および他の Microsoft アプリ、サードパーティ アプリ、カスタム アプリと統合した経験が必要となります。  
(Candidates for this exam must be able to plan, deploy, configure, and secure Teams chat, apps, channels, meetings, audio/video conferencing, live events, calling, and Microsoft Teams-certified devices. Candidates should have experience integrating Microsoft Teams with other workloads, including SharePoint, OneDrive, Exchange, Microsoft 365 Groups, Power Platform, and with other Microsoft apps, third-party apps, and custom apps.)

Microsoft Teams 管理者は、ネットワーキング、テレフォニー、ライセンス、セキュリティ、コンプライアンス、ユーザーの採用、アプリ開発、サポートなどの他の機能とも連携します。  
(A Microsoft Teams administrator also collaborates other functions, such as networking, telephony, licensing, security, compliance, user adoption, app development, and support.)

#### Microsoft Teams 環境の計画と構成 (Plan and configure a Microsoft Teams environment) (35-40%)

(analyze network usage by using Network Planner)

(specify requirements for outbound network ports and protocols for Microsoft Teams)

(determine WAN optimizer, proxy, and load balancer recommendations for Microsoft Teams)
  - Microsoft Call Quality Dashboard の Tenant Data Upload を構成する  
(configure Tenant Data Upload in Microsoft Call Quality Dashboard)
  - Microsoft Teams のレポーティング ラベルを構成する

  - QoS ポート範囲と DSCP マーキングを構成する

  - データ セキュリティに対するカスタマー ロックボックスを有効化する
