# MS-900 : Microsoft 365 Fundamentals
Microsoft 認定資格「Microsoft 365 Certified : Fundamentals」の対象試験です。
- Exam MS-900 : Microsoft 365 Fundamentals - Learn | Microsoft Docs  
https://docs.microsoft.com/learn/certifications/exams/ms-900

## 試験について (意訳)
**※2023/01/20 に改定 (アップデート) 予定です。**
- Study Guide - Exam MS-900: Microsoft 365 Fundamentals  
https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE3VwVw

この試験は、オンサイト、自宅、またはその両方の組み合わせで生産性とコラボレーションを促進するクラウドベース ソリューションの基礎レベルの知識を実証しようとしている受験者向けに設計されています。 受験者は、クラウドベースのソリューションに関する知識を持っているか、Microsoft 365 を初めて使用する可能性があります。また、この試験を使用して他の Microsoft 認定の準備をすることができますが、関連する認定の前提条件ではありません。  
(This exam is designed for candidates looking to demonstrate foundational-level knowledge of cloudbased solutions to facilitate productivity andcollaboration on-site, at home, or a combination of both. Candidates may have knowledge of cloud-based solutions or may be new to Microsoft 365. Candidates can use this exam to prepare for other Microsoft certifications, but it is not a prerequisite for any of the related certifications.)

さらに、一般的な組織の IT の課題に対処する Microsoft 365 ソリューションを推奨できる必要があります。Microsoft 365 ソリューションが生産性を向上させ、コラボレーションを促進し、コミュニケーションを最適化する方法を理解する必要があります。また、Microsoft 365 ソリューションがどのように機能するかを理解する必要があります。
データと ID を保護し、コンプライアンスを促進します。  
(Candidates should be able to recommend Microsoft 365 solutions that address common organizational IT challenges. They should understand how Microsoft 365 solutions improve productivity, facilitate collaboration, and optimize communications. They should also understand how Microsoft 365 solutions
help secure data and identity and facilitate compliance.)

加えて、エンドポイントとアプリケーションの管理、デスクトップの仮想化、自動化されたオペレーティング システムの展開、およびレポートと分析のためのソリューションを推奨できる必要があります。Microsoft 365 のライセンス、展開と移行の支援、およびクラウドへの投資を最大限に活用しようとしている組織のサポート オプションに精通している必要があります。
(Candidates should be able to recommend solutions for endpoint and application management, desktop virtualization, automated operating system deployment, and reporting and analytics. They should be familiar with Microsoft 365 licensing, deployment and migration assistance, and support options for organizations looking to maximize their investment in the cloud.)

### 対象となる領域 : 試験で評価されるスキル
試験は定期的にアップデートされ、役割を果たすために必要なスキルが反映されます。試験を受ける時期に応じて、スキル測定目標の 2 つのバージョンが含まれています。  
(Our exams are updated periodically to reflect skills that are required to perform a role. We have included two versions of the Skills Measured objectives depending on when you are taking the exam.)

常に、試験の英語版を最初にアップデートします。一部の試験は他の言語にローカライズされており、英語版がアップデートされてから約 8 週間後にアップデートされます。その他の利用可能な言語は、**試験の詳細** Web ページの**試験のスケジュール**セクションに一覧表示されています。ご希望の言語で試験を利用できない場合は、試験を完了するために 30 分の追加時間をリクエスト可可能です。  
(We always update the English language version of the exam first. Some exams are localized into other languages, and those are updated approximately eight weeks after the English version is updated. Other available languages are listed in the Schedule Exam section of the Exam Details webpage. If the exam isn't available in your preferred language, you can request an additional 30 minutes to complete the exam.)

注 : 評価される各スキル続いて記載している箇条書きの一覧は、スキルの評価項目 (どのように評価しているかを示す) となります。関連するトピックが試験内でカバーされている亜愛があります。
(Note : The bullets that follow each of the skills measured are intended to illustrate how we are assessing that skill. Related topics may be covered in the exam.)

注 : ほとんどの問題は、GA (General Availability) となった機能に関するものです。プレビュー機能が一般的に私用されている場合、それらに関する問題が含まれる場合があります。
(Note : Most questions cover features that are general availability (GA). The exam may contain questions on Preview features if those features are commonly used.)

### 機能グループ
#### クラウドの概念を説明する (Describe cloud concepts) (5–10%)
- 利用可能なさまざまな種類のクラウド サービスについて説明する  
  (Describe the different types of cloud services available)
  - Microsoft SaaS、IaaS、および PaaS の概念とユース ケースについて説明する  
    (Describe Microsoft SaaS, IaaS, and PaaS concepts and use cases)
  - Office 365 と Microsoft 365 の違いについて説明する  
    (Describe differences between Office 365 and Microsoft 365)
- クラウド、ハイブリッド、またはオンプレミスのサービスを利用する利点と考慮事項について説明する  
  (Describe the benefits of and considerations for using cloud, hybrid, or onpremises services)
  - パブリック、プライベート、およびハイブリッド クラウド モデルについて説明する  
    (Describe public, private, and hybrid cloud models)
  - クラウド、ハイブリッド、オンプレミス サービスのコストと利点を比較する  
    (Compare costs and advantages of cloud, hybrid, and on-premises services)
  - ハイブリッド ワークとフレキシブル ワークの概念について説明する  
    (Describe the concept of hybrid work and flexible work)
#### Microsoft 365 アプリとサービスを説明する (Describe Microsoft 365 apps and services) (45–50%)
- Microsoft 365 の生産性ソリューションについて説明する  
  (Describe productivity solutions of Microsoft 365)
  - Microsoft Outlook と Microsoft Exchange、Microsoft 365 アプリ、OneDrive を含む、Microsoft 365 の主要な生産性機能と利点について説明する  
    (Describe the core productivity capabilities and benefits of Microsoft 365 including Microsoft Outlook and Microsoft Exchange, Microsoft 365 apps, and OneDrive)
  - Microsoft Word、Excel、PowerPoint、Outlook、OneNote を含む、主要な Microsoft 365 アプリについて説明する  
    (Describe core Microsoft 365 Apps including Microsoft Word, Excel, PowerPoint, Outlook, and OneNote)
  - Microsoft Project、Planner、予約、フォーム、リスト、To Do を含む、Microsoft 365 の作業管理機能について説明する  
    (Describe work management capabilities of Microsoft 365 including Microsoft Project, Planner, Bookings, Forms, Lists, and To Do)
- Microsoft 365 のコラボレーション ソリューションについて説明する  
  (Describe collaboration solutions of Microsoft 365)
  - Microsoft Exchange、Outlook、Yammer、SharePoint、OneDrive、Stream を含む、Microsoft 365 のコラボレーションのメリットと機能について説明する  
    (Describe the collaboration benefits and capabilities of Microsoft 365 including Microsoft Exchange, Outlook, Yammer, SharePoint, OneDrive, and Stream)
  - Microsoft Teams と Teams Phone のコラボレーションの利点と機能について説明する  
    (Describe the collaboration benefits and capabilities of Microsoft Teams and Teams Phone)
  - Microsoft Viva アプリについて説明する  
    (Describe the Microsoft Viva apps)
  - コラボレーション アプリを用いて Microsoft Teams を拡張する方法を説明する  
    (Describe the ways that you can extend Microsoft Teams by using collaborative apps)
- Microsoft 365 でのエンドポイントの最新化、管理の概念、展開オプションについて説明する  
  (Describe endpoint modernization, management concepts, and deployment options in Microsoft 365)
  - Microsoft エンドポイント マネージャー (MEM)、Intune、AutoPilot、SCCM との共同管理、テナント アタッチを含む、Microsoft 365 のエンドポイント管理機能について説明する
    (Describe the endpoint management capabilities of Microsoft 365 including Microsoft Endpoint Manager (MEM), Intune, AutoPilot, co-management with SCCM, and tenant attach)
  - Windows 365 と Azure Virtual Desktop の違いを比較する  
    (Compare the differences between Windows 365 and Azure Virtual Desktop)
  - 展開リングを含む Windows-as-a-Service (WaaS) の展開モデルとリリース モデルについて説明する  
    (Describe the deployment and release models for Windows-as-a-Service (WaaS) including deployment rings)
  - Microsoft 365 アプリの展開とアップデートのチャネルを特定する
    (Identify deployment and update channels for Microsoft 365 Apps)
- Microsoft 365 の分析機能について説明する  
  (Describe analytics capabilities of Microsoft 365)
  - Viva Insights の機能について説明する  
    (Describe the capabilities of Viva Insights)
  - Microsoft 365 管理センターと Microsoft 365 ユーザー ポータルの機能について説明する  
    (Describe the capabilities of the Microsoft 365 Admin center and Microsoft 365 user portal)
  - Microsoft 365 管理センターおよびその他の管理センターで利用できるレポートについて説明する  
    (Describe the reports available in the Microsoft 365 Admin center and other admin centers)
#### Microsoft 365 のセキュリティ、コンプライアンス、プライバシー、信頼を説明する (Describe security, compliance, privacy, and trust in Microsoft 365) (25–30%)
- Microsoft 365 のゼロ トラスト セキュリティ原則について説明する  
  (Describe Zero Trust security principles for Microsoft 365)
  - ゼロ トラスト モデルについて説明する  
    (Describe the Zero Trust model)
  - Microsoft Granular Delegated Admin Privileges (GDAP) の原則について説明する  
    (Describe Microsoft Granular Delegated Admin Privileges (GDAP) principles)
- Microsoft 365 の ID およびアクセス管理ソリューションについて説明する  
  (Describe identity and access management solutions of Microsoft 365)
  - Microsoft Entra の一部である Azure Active Directory と Azure Identity を含む、Microsoft 365 の ID およびアクセス管理機能について説明する  
    (Describe the identity and access management capabilities of Microsoft 365 including Azure Active Directory, part of Microsoft Entra, and Azure Identity)
  - クラウド ID、オンプレミス ID、およびハイブリッド ID の概念について説明する  
    (Describe cloud identity, on-premises identity, and hybrid identity concepts)
  - 多要素認証 (MFA) とセルフサービス パスワード リセット (SSPR) を実装する目的と価値を説明する  
    (Describe the purpose and value of implementing multi-factor authentication (MFA) and selfservice password reset (SSPR))
  - 条件付きアクセスの目的と価値について説明する  
    (Describe the purpose and value of conditional access)
- Microsoft 365 の脅威保護ソリューションについて説明する  
  (Describe threat protection solutions of Microsoft 365)
  - Microsoft 365 Defender、Defender for Endpoint、Defender for Office 365、Defender for Identity、Defender for Cloud Apps、および Microsoft 365 Defender ポータルについて説明する  
    (Describe Microsoft 365 Defender, Defender for Endpoint, Defender for Office 365, Defender for Identity, Defender for Cloud Apps, and the Microsoft 365 Defender Portal)
  - Microsoft Secure Score の利点と機能について説明する  
    (Describe Microsoft Secure Score benefits and capabilities)
  - エンドポイント、アプリケーション、ID に対する最も一般的な種類の脅威について説明する  
    (Describe the most common types of threats against endpoints, applications, and identities)
  - Microsoft 365 が最も一般的な脅威に対処する方法について説明する  
    (Describe how Microsoft 365 addresses the most common threats)
  - Microsoft Sentinel と Microsoft 365 Lighthouse の機能と利点について説明する  
    (Describe the capabilities and benefits of Microsoft Sentinel and Microsoft 365 Lighthouse)
- Microsoft 365 の信頼、プライバシー、リスク、およびコンプライアンス ソリューションについて説明する  
  (Describe trust, privacy, risk, and compliance solutions of Microsoft 365)
  - Microsoft Purview のリスクとコンプライアンス ソリューション、および Microsoft 365 のコンプライアンスについて説明する  
    (Describe Microsoft Purview risk and compliance solutions and compliance in Microsoft 365)
  - Microsoft がデータ レジデンシーをサポートして規制遵守を確保する方法を説明する  
    (Describe how Microsoft supports data residency to ensure regulatory compliance)
  - 情報保護機能について説明する  
    (Describe information protection features)
  - Microsoft Priva の機能と利点を説明する  
    (Describe the capabilities and benefits of Microsoft Priva)
  - 内部の脅威から保護するためのインサイダー リスク管理ソリューションについて説明する  
    (Describe insider risk management solutions to protect against internal threats)
  - 監査と電子情報開示ソリューションについて説明する  
    (Describe auditing and eDiscovery solutions)
#### Microsoft 365 の価格、ライセンス、サポートを説明する (Describe Microsoft 365 pricing, licensing, and support) (10–15%)
- Microsoft 365 の価格設定と請求管理オプションを特定する  
  (Identify Microsoft 365 pricing and billing management options)
  - エンタープライズ契約、クラウド ソリューション プロバイダー、直接請求を含む、マイクロソフト クラウド サービスの価格モデルについて説明する  
    (Describe the pricing model for Microsoft cloud services including enterprise agreements, cloud solution providers, and direct billing)
  - 請求頻度や支払い方法を含む、利用可能な請求および請求管理オプションについて説明する  
    (Describe available billing and bill management options including billing frequency and methods of payment)
- Microsoft 365 で利用可能なライセンス オプションを特定する  
  (Identify licensing options available in Microsoft 365)
  - ライセンス管理について説明する  
    (Describe license management)
  - 基本ライセンスとアドオン ライセンスの違いについて説明する  
    (Describe the differences between base licensing and add-on licensing)
- Microsoft 365 サービスのサポート オプションを特定する  
  (Identify support options for Microsoft 365 services)
  - Microsoft 365 サービスのサポート リクエストを作成する方法を説明する  
    (Describe how to create a support request for Microsoft 365 services)
  - Microsoft 365 サービスのサポート オプションについて説明する  
    (Describe support options for Microsoft 365 services)
  - サービス クレジットを含むサービス レベル アグリーメント (SLA) について説明する  
    (Describe service level agreements (SLAs) including service credits)
  - Microsoft 365 管理ポータル、もしくは Azure テナント ポータルを用いて、サービスの正常性状態を確認する  
    (Determine service health status by using the Microsoft 365 admin portal or the Azure tenant portal)
