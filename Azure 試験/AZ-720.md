### 試験の対象者
Azure Support Engineer for Connectivity Specialty 認定の受験者は、高度なトラブルシューティング方法を用いて Azure ネットワークと接続の問題を解決する専門知識を持つサポートエンジニアです。

Professionals in this role troubleshoot hybrid environments, including issues with Azure Virtual Machines, virtual networks, and connectivity between on-premises and Azure services. They use various tools and technologies to diagnose and identify root causes for complex issues.

Azure 仮想マシン、仮想ネットワーク、オンプレミスと Azure サービス間の接続に関する問題を含むハイブリッド環境のトラブルシューティングを行います。さまざまなツールとテクノロジーを使用して、複雑な問題の根本原因を診断および特定します。

またルーティング、権限、アカウント制限の知識を含むネットワークおよびハイブリッド環境の経験が必要となります。利用可能なツールを用いて、ビジネス継続性、ハイブリッド環境、IaaS (Infrastructure as a Service)、PaaS (Platform as a Service)、アクセス制御、ネットワーキング、および仮想マシンの接続に関連する問題を診断できる必要があります。

### スキルの評価
注：評価される各スキルの下に記載しているリストは、スキルの評価項目です。このリストは、決定的でも網羅的でもありません。

注：試験内容について、ほとんどの場合、プレビュー機能はカバーされず、一部の機能は GA (General Availability) となった際に試験に追加されます。

#### ビジネス継続姓の問題に関するトラブルシュート (5–10%)
- バックアップの問題をトラブルシュートする
  - バックアップ ログをレビュー、解釈する
  - 失敗したバックアップ ジョブの再開を含む Azure 仮想マシンのバックアップの問題をトラブルシュートする
  - Azure Backup エージェントの問題をトラブルシュートする
  - Azure Backup Server 問題をトラブルシュートする
  - スケジュールされたバックアップをトラブルシュートする
- リカバリの問題をトラブルシュートする
  - Azure Site Recovery の問題をトラブルシュートする
  - Hyper-V、VMware ESX、もしくは System Center Configuration Manager を含むハイブリッド シナリオのサイト リカバリをトラブルシュートする
  - Azure Backup Agent、Azure backup、もしくは Azure Backup Server を用いた場合のリストア問題をトラブルシュートする
  - Azure 仮想マシン バックアップからリカバリするファイルの問題をトラブルシュートする
#### ハイブリットおよびクラウド接続性の問題に関するトラブルシュート (20–25%)
- 仮想ネットワーク (VNet) の接続性をトラブルシュートする
  - 仮想プライベート ネットワーク (VPN) ゲートウェイ トランジット問題をトラブルシュートする
  - ハブ アンド スポーク VNet 構成問題をトラブルシュートする
  - グローバル VNet ピアリングの接続性問題をトラブルシュートする
  - ピアリングされた接続をトラブルシュートする
- 名前解決問題をトラブルシュートする
  - Azure が提供する名前解決を用いたシナリオの名前解決をトラブルシュートする
  - カスタム DNS サーバーを用いたシナリオの名前解決をトラブルシュートする
  - DNS 監査ログをレビュー、解釈する
  - Azure プライベート DNS ゾーンの名前解決をトラブルシュートする
  - パブリック DNS プロバイダの DNS レコード問題をトラブルシュートする
  - ドメインの委任問題をトラブルシュートする
- Point-to-Site 仮想プライベート ネットワーク (VPN) の接続性をトラブルシュートする
  - Windows VPN クライアント構成問題をトラブルシュートする
  - OpenVPN VPN クライアント構成問題をトラブルシュートする
  - macOS VPN クライアント構成問題をトラブルシュートする
  - 証明書ベースの VPN 接続問題をトラブルシュートする
  - RADIUS ベースの VPN 接続問題をトラブルシュートする
  - Azure Active Directory (Azure AD) 認証問題をトラブルシュートする
- Site-to-Site 仮想プライベート ネットワークの接続性問題をトラブルシュートする
  - VPN ゲートウェイからネットワーク ログとキャプチャされたネットワーク トラフィックをレビュー、解釈する
  - Site-to-Site VPN のレイテンシ問題に対する根本原因を決定する
  - ゲートウェイ構成スクリプトをレビュー、解釈する
  - VPN ゲートウェイをリセットする
  - 実行中の Log Analytics クエリのゲートウェイ問題をトラブルシュートする
- Azure ExpressRoute 接続性の問題をトラブルシュートする
  - ルートが有効で正しく構成されているかどうかを決定する
  - ExpressRoute 回線に対するピアリング構成を検証する
  - ExpressRoute 回線をリセットする
  - ルート フィルタリングをトラブルシュートする
  - カスタム定義ルートをトラブルシュートする
  - ExpressRoute に関連したレイテンシ問題の根本原因を決定する
#### Platform as a Service の問題に関するトラブルシュート (5–10%)
- PaaS サービスをトラブルシュートする
  - PaaS への接続問題をトラブルシュートする
  - PaaS サービスのファイアウォールをトラブルシュートする
  - PaaS 構成問題をトラブルシュートする
  - サービスレベル スロットリングの根本的な原因を決定する
- PaaS の統合問題をトラブルシュートする
  - 仮想ネットワークを持つ PaaS サービスとの統合問題をトラブルシュートする
  - サブネット委任の問題をトラブルシュートする
  - プライベート エンドポイントとサービス エンドポイントの問題をトラブルシュートする
  - Azure プライベート リンクの問題をトラブルシュートする
#### 認証およびアクセス制御の問題に関するトラブルシュート (15–20%)
- Azure AD 認証をトラブルシュートする
  - オンプレミス システムがなぜ Azure リソースに接続できないかを決定する
  - Azure AD 構成の問題をトラブルシュートする
  - セルフサービス パスワード リセット問題をトラブルシュートする
  - 多要素認証の問題をトラブルシュートする
- ハイブリッド認証をトラブルシュートする
  - Azure AD Connect 同期問題をトラブルシュートする
  - Azure AD をActive Directory Domain Services (Azure AD DS) へ統合する問題をトラブルシュートする
  - Azure AD と Active Directory Federation Services (AD FS) 間の接続性問題をトラブルシュートする
  - パススルー認証とパスワード ハッシュ同期の問題をトラブルシュートする
  - AzureADアプリケーション プロキシ接続の問題をトラブルシュートする
- 認証の問題をトラブルシュートする
  - ロールベースのアクセス制御 (RBAC) 問題をトラブルシュートする
  - Azure Key Vault に格納された暗号化パスワードの問題をトラブルシュートする
  - Azure AD 条件付きアクセス ポリシーに関連したサインイン問題をトラブルシュートする
#### ネットワークに関するトラブルシュート (25–30%)
- Azure ネットワーク セキュリティ問題をトラブルシュートする
  - Azure Web Application Firewall がなぜトラフィックをブロックしたのかを決定する
  - Point-to-Site や Site-to-Site シナリオに対する暗号化および証明書問題をトラブルシュートする
  - 安全なエンドポイントへの接続をトラブルシュートする
- Azure ネットワーク セキュリティ グループ (NSG) をトラブルシュートする
  - NSG 構成問題をトラブルシュートする
  - NSG フロー ログをレビュー、解釈する
  - VM,もしくは VM グループがアプリケーション セキュリティ グループ (ASG) に関連付けられているかどうかを決定する
- Azure Firewall 問題をトラブルシュートする
  - アプリケーション、ネットワーク、およびインフラストラクチャ ルールをトラブルシュートする
  - ネットワーク アドレス変換 (NAT) や分散ネットワーク アドレス変換 (DNAT) ルールをトラブルシュートする
  - Azure Firewall Manager 構成問題をトラブルシュートする
- レイテンシ問題をトラブルシュートする
  - VM レベルのスロットリングに対する根本原因を決定する
  - Azure 仮想マシンに接続する際のレイテンシ問題に対する根本原因を決定する
  - ソース リソースと宛先リソース間のスロットリングに対する根本原因を決定する
  - 帯域幅の可用性問題をトラブルシュートする
  - リソースの応答時間がサービスレベル契約 (SLA) を満たしているかどうかを決定する
- ルーティングとトラフィック制御をトラブルシュートする
  - ルート テーブルをレビュー、解釈する
  - 非対称ルーティングをトラブルシュートする
  - ユーザー定義ルートの問題をトラブルシュートする
  - 強制トンネリングに関連する問題をトラブルシュートする
  - Border Gateway Protocol (BGP) 問題をトラブルシュートする
  - 仮想ネットワーク ピアリング、推移的ルーティング、およびサービス チェーンをトラブルシュートする
  - Azure のルーティング構成問題をトラブルシュートする
- 負荷分散の問題をトラブルシュートする
  - 負荷分散されたクラスター内の VM が正常かどうかを決定する
  - Azure Load Balancer の問題をトラブルシュートする
  - 負荷分散ルールをレビュー、解釈する
  - トラフィック分散問題をトラブルシュートする
  - Azure Traffic Manager の構成を評価する
  - Azure Traffic Manager プロファイルの問題をトラブルシュートする
  - ポート枯渇の問題をトラブルシュートする
  - Azure Front Door の問題をトラブルシュートする
  - Azure Application Gateway の問題をトラブルシュートする
#### VM 接続性問題に関するトラブルシュート (5–10%)
- Azure Bastion をトラブルシュートする
  - Azure Bastion のデプロイ問題をトラブルシュートする
  - 接続性問題をトラブルシュートする
  - 承認に関する問題をトラブルシュートする
- ジャストインタイム (JIT) VM アクセスをトラブルシュートする
  - VM の接続性を検証する
  - Microsoft Defender for Cloud 構成問題をトラブルシュートする
  - JIT VM アクセスの使用を許可されているリソースを決定する
