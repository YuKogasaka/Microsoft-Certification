# AZ-900 : Microsoft Azure Fundamentals
Microsoft 認定資格「Microsoft Certified : Azure Fundamentals」の対象試験です。
- Exam AZ-900 : Microsoft Azure Fundamentals - Learn | Microsoft Docs  
https://docs.microsoft.com/ja-jp/learn/certifications/exams/az-900

## 評価されるスキルについて (意訳)
**※2022/10/28 に英語版が改定 (アップデート) されました。**
- Study Guide - Exam AZ-900: MicrosoftAzure Fundamentals  
https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE3VwUY

この試験の受験者は、クラウドの概念と Microsoft Azure に関する基本的な知識を証明したいテクノロジー プロフェッショナルです。  
(Candidates for this exam are technology professionals who want to demonstrate foundational knowledge of cloud concepts and Microsoft Azure.)

また、コンピューティング、ネットワーク、ストレージなどの Azure アーキテクチャ コンポーネントと Azure サービスについて説明できる必要があります。加えて、Azure を保護、管理、および管理するための機能とツールについても説明できる必要があります。  
(These professionals can describe Azure architectural components and Azure services, such as compute, networking, and storage. They can also describe features and tools to secure, govern, and administer Azure.)

さらに、インフラストラクチャ管理、データベース管理、ソフトウェア開発など、情報技術の分野で働いたスキルと経験が必要となります。  
(Candidates for this exam have skills and experience working with an area of information technology, such as infrastructure management, database management, or software development.)

試験は定期的にアップデートされ、役割を果たすために必要なスキルが反映されます。試験を受ける時期に応じて、スキル測定目標の 2 つのバージョンが含まれています。  
(Our exams are updated periodically to reflect skills that are required to perform a role. We have included two versions of the Skills Measured objectives depending on when you are taking the exam.)

Other available languages are listed in the Schedule Exam section of the Exam Details webpage. If the exam isn't available in your preferred language, you can request an additional 30 minutes to complete the exam.)

(Note : Most questions cover features that are general availability (GA). The exam may contain questions on Preview features if those features are commonly used.)

### 機能グループ
#### クラウドの概念を説明する (Describe cloud concepts) (25–30%)
- クラウド コンピューティングを説明する  
(Describe cloud computing)
  - クラウド コンピューティングを定義する  
  (Define cloud computing)
  - 共有責任モデルを説明する  
  (Describe the shared responsibility model
  - パブリック、プライベート、ハイブリッドを含む、クラウド モデルを定義する  
  (Define cloud models, including public, private, and hybrid)
  - 各クラウド モデルに対する適切なユース ケースを特定する  
  (Identify appropriate use cases for each cloud model)
  - 消費量ベース モデルを説明する  
  (Describe the consumption-based model)
  - クラウドの価格モデルを比較する  
  (Compare cloud pricing models)
- クラウド サービスを用いたメリットを説明する  
(Describe the benefits of using cloud services)
  - クラウドにおける高可用性とスケーラビリティのメリットについて説明する  
  (Describe the benefits of high availability and scalability in the cloud)
  - クラウドにおける信頼性や予測可能性のメリットを説明する  
  (Describe the benefits of reliability and predictability in the cloud)
  - クラウドにおけるセキュリティやガバナンスのメリットを説明する  
  (Describe the benefits of security and governance in the cloud)
  - クラウドにおける管理性のメリットを説明する  
  (Describe the benefits of manageability in the cloud)
- クラウド サービスのタイプを説明する  
(Describe cloud service types)
  - Infrastructure-as-a-Service (IaaS) を説明する  
  (Describe infrastructure as a service (IaaS))
  - Platform-as-a-Service (PaaS) を説明する  
  (Describe platform as a service (PaaS))
  - Software-as-a-Service (SaaS) を説明する  
  (Describe software as a service (SaaS))
  - それぞれのクラウド サービス (IaaS, PaaS, SaaS) に対する適切なユース ケースを特定する  
  (Identify appropriate use cases for each cloud service (IaaS, PaaS, SaaS))
#### Azure のアーキテクチャとサービスを説明する (Describe Azure architecture and services) (35–40%)
- Azure のコア アーキテクチャ コンポーネントを説明する  
(Describe the core architectural components of Azure)
  - Azure リージョン、リージョン ペア、ソブリン リージョンを説明する  
  (Describe Azure regions, region pairs, and sovereign regions)
  - 可溶性ゾーンを説明する  
  (Describe availability zones)
  - Azure データセンターを説明する  
  (Describe Azure datacenters)
  - Azure リソースとリソース グループを説明する  
  (Describe Azure resources and resource groups)
  - サブスクリプションを説明する  
  (Describe subscriptions)
  - 管理グループを説明する  
  (Describe management groups)
  - リソース グループ、サブスクリプション、および管理グループの階層を説明する  
  (Describe the hierarchy of resource groups, subscriptions, and management groups)
- Azure Compute と ネットワーキング サービスを説明する  
(Describe Azure compute and networking services)
  - コンテナー インスタンス、仮想マシン (VM)、Functions を含む Compute のタイプを比較する  
  (Compare compute types, including container instances, virtual machines (VMs), and functions)
  - Azure Virtual Machines、Azure Virtual Machine スケールセット、可溶性セット、Azure Virtual Desktop を含む VM オプションを説明する  
  (Describe VM options, including Azure Virtual Machines, Azure Virtual Machine Scale Sets, availability sets, and Azure Virtual Desktop)
  - 仮想マシンに必要なリソースを説明する  
  (Describe resources required for virtual machines)
  - Azure App Service、コンテナー、仮想マシンの Web アプリ機能を含む、アプリケーション ホスティング プランを説明する  
  (Describe application hosting options, including the Web Apps feature of Azure App Service, containers, and virtual machines)
  - Azure Virtual Networks の目的、Azure 仮想サブネット、ピアリング、Azure DNS、Azure VPN Gateway、Azure ExpressRoute を含む、仮想ネットワークを説明する  
  (Describe virtual networking, including the purpose of Azure Virtual Networks, Azure virtual subnets, peering, Azure DNS, Azure VPN Gateway, and Azure ExpressRoute)
  - パブリック/プライベート エンドポイントを定義する  
  (Define public and private endpoints)
- Azure ストレージ サービスを説明する  
(Describe Azure storage services)
  - Azure ストレージ サービスを比較する  
  (Compare Azure storage services)
  - ストレージの層を説明する  
  (Describe storage tiers)
  - 冗長性オプションを説明する  
  (Describe redundancy options)
  - ストレージ アカウント オプションやストレージのタイプを説明する  
  (Describe storage account options and storage types)
  - AzCopy、Azure Storage Explorer、Azure File Sync を含むファイルの移動オプションを特定する  
  (Identify options for moving files, including AzCopy, Azure Storage Explorer, and Azure File Sync)
  - Azure Migrate や Azure Data Box を含む移行オプションを説明する  
  (Describe migration options, including Azure Migrate and Azure Data Box)
- Azure の ID、アクセス、セキュリティを説明する  
(Describe Azure identity, access, and security)
  - Microsoft Entra の一部である Microsoft Azure Active Directory (Azure AD) や Azure Active Directory Domain Services (Azure AD DS) を含む Azure のドメイン サービスを説明する  
  (Describe directory services in Azure, including Microsoft Azure Active Directory (Azure AD), part of Microsoft Entra and Azure Active Directory Domain Services (Azure AD DS))
  - シングル サインオン (SSO)、多要素認証、およびパスワードレスを含む Azure の認証方法を説明する  
  (Describe authentication methods in Azure, including single sign-on (SSO), multifactor authentication, and passwordless)
  - Azure の外部 ID とゲスト アクセスを説明する  
  (Describe external identities and guest access in Azure)
  - Microsoft Entra の一部である Azure Active Directory (Azure AD) の条件付きアクセスを説明する  
  (Describe Conditional Access in Microsoft Azure Active Directory (Azure AD), part of Microsoft Entra)
  - ロールベースのアクセス制御 (RBAC) を説明する  
  (Describe Azure role-based access control (RBAC))
  - ゼロ トラストの概念を説明する  
  (Describe the concept of Zero Trust)
  - 多層防御モデルの目的を説明する  
  (Describe the purpose of the defense in depth model)
  - Microsoft Defender for Cloud の目的を説明する  
  (Describe the purpose of Microsoft Defender for Cloud)
#### Azure の管理とガバナンスを説明する (Describe Azure management and governance) (30–35%)
- Azure のコスト管理を説明する  
(Describe cost management in Azure)
  - Azure のコストに影響する要素を説明する  
(Describe factors that can affect costs in Azure)
  - Pricing calculator と Total Cost of Ownership (TCO) calculator を比較する  
  (Compare the Pricing calculator and the Total Cost of Ownership (TCO) calculator)
  - Azure Cost Management and Billing ツールを説明する  
  (Describe the Azure Cost Management and Billing tool)
  - タグの目的を説明する  
  (Describe the purpose of tags)
- Azure のガバナンスやコンプライアンスに対する機能やツールを説明する  
(Describe features and tools in Azure for governance and compliance)
  - Azure Blueprints の目的を説明する  
  (Describe the purpose of Azure Blueprints)
  - Azure Policy の目的を説明する  
  (Describe the purpose of Azure Policy)
  - リソース ロックの目的を説明する  
  (Describe the purpose of resource locks)
  - Service Trust Portal の目的を説明する  
  (Describe the purpose of the Service Trust Portal)
- Azure リソースの管理やデプロイするための機能とツールを説明する  
(Describe features and tools for managing and deploying Azure resources)
  - Azure ポータルを説明する  
  (Describe the Azure portal)
  - Azure CLI や Azure PowerShell を含む Azure Cloud Shell を説明する  
  (Describe Azure Cloud Shell, including Azure CLI and Azure PowerShell)
  - Azure Arc の目的を説明する  
  (Describe the purpose of Azure Arc)
  - Azure Resource Manager や Azure Resource Manager テンプレート (ARM テンプレート) を説明する  
  (Describe Azure Resource Manager and Azure Resource Manager templates (ARM templates))
- Azure の監視ツールを説明する  
(Describe monitoring tools in Azure)
  - Azure Advisor の目的を説明する  
  (Describe the purpose of Azure Advisor)
  - Azure Service Health を説明する  
  (Describe Azure Service Health)
  - Log Analytics、Azure Monitor アラート、および Application Insights を含む Azure Monitor を説明する  
  (Describe Azure Monitor, including Log Analytics, Azure Monitor alerts, and Application Insights)
