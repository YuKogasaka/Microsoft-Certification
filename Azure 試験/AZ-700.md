## 試験について (意訳)
**※2022/11/01 に改定 (アップデート) される予定です。**
- Study Guide - Exam AZ-700: Designing and Implementing Microsoft Azure Networking Solutions  
https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE4PaHw

この試験の受験者は、ハイブリッド ネットワーキング、接続、ルーティング、セキュリティ、Azure サービスへのプライベート アクセスを含む、Azure ネットワーキング ソリューションの計画、実装、保守に関する専門知識を持っている必要があります。  
(Candidates for this exam should have subject matter expertise in planning, implementing, and maintaining Azure networking solutions, including hybrid networking, connectivity, routing, security, and private access to Azure services.)

Azure ネットワーク エンジニアの責任には、Azure ネットワーク ソリューションの推奨、計画、および実装が含まれます。このロールの専門家は、ソリューションのパフォーマンス、復元力、規模、およびセキュリティを管理します。Azure Portal や PowerShell、Azure コマンド ライン インターフェイス (CLI)、Azure Resource Manager テンプレート (ARM テンプレート) を含む他の方法を使用して、ネットワーク ソリューションをデプロイします。  
(Responsibilities for the Azure Network Engineer include recommending, planning, and implementing Azure networking solutions. Professionals in this role manage the solution for performance, resiliency, scale, and security. They deploy networking solutions by using the Azure Portal and other methods, including PowerShell, Azure Command-Line Interface (CLI), and Azure Resource Manager templates (ARM templates).)

Azure ネットワーク エンジニアは、ソリューション アーキテクト、クラウド管理者、セキュリティ エンジニア、アプリケーション開発者、および DevOps エンジニアと協力して、Azure ソリューションを提供します。  
(The Azure Network Engineer works with solution architects, cloud administrators, security engineers, application developers, and DevOps engineers to deliver Azure solutions.)

また、ネットワーク、ハイブリッド接続、およびネットワーク セキュリティに関する豊富な経験と知識に加えて、専門的な Azure 管理スキルを持っている必要があります。  
(Candidates for this exam should have expert Azure administration skills, in addition to extensive experience and knowledge of networking, hybrid connections, and network security)

### 機能グループ
#### ハイブリッド ネットワークの設計、実装と管理 (Design, implement, and manage hybrid networking) (10–15%)
- Site-to-Site VPN 接続を設計、実装、管理する  
(Design, implement, and manage a site-to-site VPN connection)
  - 高可用性の Site-to-Site VPN 接続を設計する  
  (Design a site-to-site VPN connection for high availability)
  - 適切な仮想ネットワーク (VNet) ゲートウェイ SKU を選択する  
  (Select an appropriate virtual network (VNet) gateway SKU)
  - ポリシー ベース VPN や ルート ベース VPN をいつ利用するかを特定する  
  (Identify when to use policy-based VPN versus route-based VPN)
  - ローカル ネットワーク ゲートウェイを作成、構成する  
  (Create and configure a local network gateway)
  - IPsec/IKE ポリシーを作成、構成する  
  (Create and configure an IPsec/IKE policy)
  - 仮想ネットワーク ゲートウェイを作成、構成する  
  (Create and configure a virtual network gateway)
  - 仮想ネットワーク ゲートウェイとの接続性の問題を診断、特定する  
  (Diagnose and resolve virtual network gateway connectivity issues)
- Point-to-Site VPN 接続を設計、実装、管理する  
(Design, implement, and manage a point-to-site VPN connection)
  - 適切な仮想ネットワークゲートウェイ SKU を選択する  
  (Select an appropriate virtual network gateway SKU)
  - RADIUS 認証を計画、構成する  
  (Plan and configure RADIUS authentication)
  - 証明書ベース認証を計画、構成する  
  (Plan and configure certificate-based authentication)
  - OpenVPN 認証を計画、構成する  
  (Plan and configure OpenVPN authentication)
  - Microsoft Entra の一部である Azure Active Directory (Azure AD) 認証を計画、構成する  
  (Plan and configure authentication by using Microsoft Azure Active Directory (Azure AD), part of Microsoft Entra)
  - VPN クライアント構成ファイルを実装する  
  (Implement a VPN client configuration file)
  - クライアントサイドと認証の問題を診断、解決する  
  (Diagnose and resolve client-side and authentication issues)
- Azure ExpressRoute を設計、実装、管理する  
(Design, implement, and manage Azure ExpressRoute)
  - プロバイダー モデルとダイレクト モデル (ExpressRoute Direct) を選択する  
  (Choose between provider and direct model (ExpressRoute Direct))
  - 複数の ExpressRoute の場所間の Azure クロスリージョン接続を設計、実装する  
  (Design and implement Azure cross-region connectivity between multiple ExpressRoute locations)
  - 適切な ExpressRoute SKU と層を選択する  
  (Select an appropriate ExpressRoute SKU and tier)
  - ExpressRoute Global Reach を設計、実装する  
  (Design and implement ExpressRoute Global Reach)
  - ExpressRoute FastPath を設計、実装する  
  (Design and implement ExpressRoute FastPath)
  - プライベート ピアリングのみ、Microsoft ピアリングのみ、もしくは両方を選択する  
  (Choose between private peering only, Microsoft peering only, or both)
  - プライベート ピアリングを構成する  
  (Configure private peering)
  - Microsoft ピアリングを構成する  
  (Configure Microsoft peering)
  - ExpressRoute ゲートウェイを作成、構成する  
  (Create and configure an ExpressRoute gateway)
  - 仮想ネットワークを ExpressRoute 回線に接続する  
  (Connect a virtual network to an ExpressRoute circuit)
  - ルート アドバタイズ構成を推奨する  
  (Recommend a route advertisement configuration)
  - ExpressRoute を介した暗号化を構成する  
  (Configure encryption over ExpressRoute)
  - 双方向フォワーダ検出 (Bidirectional Forwarding Detection) を実装する  
  (Implement Bidirectional Forwarding Detection)
  - ExpressRoute 接続の問題を診断、解決する  
  (Diagnose and resolve ExpressRoute connection issues)
#### コア ネットワーク インフラストラクチャの設計と実装 (Design and implement core networking infrastructure) (20–25%)
- VNet にプライベート IP アドレスを設計、実装する  
(Design and implement private IP addressing for VNets)
  - VNet を作成する  
  (Create a VNet)
  - VNet ゲートウェイ、プライベート エンドポイント ファイアウォール、アプリケーション ゲートウェイ、VNet 統合プラットフォーム サービスを含むサービスのサブネット化を計画、構成する  
  (Plan and configure subnetting for services, including VNet gateways, private endpoints, firewalls, application gateways, and VNet-integrated platform services)
  - サブネットの委任を計画、構成する  
  (Plan and configure subnet delegation)
  - Azure Route Server のサブネットを計画、構成する  
  (Plan and configure subnetting for Azure Route Server)
- 名前解決を設計、実装する  
(Design and implement name resolution)
  - パブリック DNS ゾーンを設計する  
  (Design public DNS zones)
  - プライベート DNS ゾーンを設計する  
  (Design private DNS zones)
  - VNet 内に名前解決を設計する  
  (Design name resolution inside a VNet)
  - パブリック/プライベート DNS ゾーンを構成する  
  (Configure a public or private DNS zone)
  - プライベート DNS ゾーンを VNet にリンクする  
  (Link a private DNS zone to a VNet)
- クロス VNet 接続を設計、実装する  
(Design and implement cross-VNet connectivity)
  - ゲートウェイ トランジットを含むサービス チェーンを設計する  
  (Design service chaining, including gateway transit)
  - VNet 間の VPN 接続を設計する  
  (Design VPN connectivity between VNets)
  - VNet ピアリングを実装する  
  (Implement VNet peering)
- Azure Virtual WAN アーキテクチャを設計、実装する  
(Design and implement an Azure Virtual WAN architecture)
  - 選択するタイプとサービスを含む Azure Virtual WAN アーキテクチャを設計する  
  (Design an Azure Virtual WAN architecture, including selecting types and services)
  - VNet ゲートウェイを Azure Virtual WAN に接続する  
  (Connect a VNet gateway to Azure Virtual WAN)
  - Virtual WAN 内にハブを作成する  
  (Create a hub in Virtual WAN)
  - 仮想ハブのネットワーク仮想アプライアンス (NVA) を作成する  
  (Create a network virtual appliance (NVA) in a virtual hub)
  - 仮想ハブ ルーティングを構成する  
  (Configure virtual hub routing)
  - 接続ユニットを作成する  
  (Create a connection unit)
#### ルーティングの設計と実装 (Design and implement routing) (25–30%)
- VNet ルーティングを設計、実装、管理する  
(Design, implement, and manage VNet routing)
  - ユーザー定義ルート (UDR) を設計、実装する  
  (Design and implement user-defined routes (UDRs))
  - サブネットとルート テーブルを関連付ける  
  (Associate a route table with a subnet)
  - 強制トンネリングを構成する  
  (Configure forced tunneling)
  - ルーティングの問題を診断、解決する  
  (Diagnose and resolve routing issues)
  - Azure Route Server を設計、実装する  
  (Design and implement Azure Route Server)
- Azure Load Balancer を設計、実装する  
(Design and implement an Azure Load Balancer)
  - Azure Load Balancer SKU (Basic か Standard) を選択する  
  (Choose an Azure Load Balancer SKU (Basic versus Standard))
  - パブリックか内部かを選択する  
  (Choose between public and internal)
  - (クロスリージョンを含む) Azure Load Balancer を作成、構成する  
  (Create and configure an Azure Load Balancer (including cross-region))
  - 負荷分散ルールを実装する  
  (Implement a load balancing rule)
  - インバウンド NAT ルールを作成、構成する  
  (Create and configure inbound NAT rules)
  - ロード バランサーの明示的なアウトバウンド ルールを作成する  
  (Create explicit outbound rules for a load balancer)
- Azure Application Gateway を設計、実装する  
(Design and implement Azure Application Gateway)
  - Azure Application Gateway デプロイメント オプションを推奨する  
  (Recommend Azure Application Gateway deployment options)
  - 手動か自動スケールかを選択する  
  (Choose between manual and autoscale)
  - バックエンド プールを作成する  
  (Create a back-end pool)
  - 正常性プローブを構成する  
  (Configure health probes)
  - リスナーを構成する  
  (Configure listeners
  - ルーティング ルールを構成する  
  (Configure routing rules)
  - HTTP 設定を構成する  
  (Configure HTTP settings)
  - Transport Layer Security (TLS) を構成する  
  (Configure Transport Layer Security (TLS)
  - 書き換えセットを構成する  
  (Configure rewrite sets)
- Azure Front Door を実装する  
(Implement Azure Front Door)
  - Azure Front Door SKU を選択する  
  (Choose an Azure Front Door SKU)
  - HTTP レスポンス コードのカスタマイズを含む正常性プローブを構成する  
  (Configure health probes, including customization of HTTP response codes)
  - SSL 終了とエンドツーエンド SSL 暗号化を構成する  
  (Configure SSL termination and end-to-end SSL encryption)
  - マルチサイト リスナーを構成する  
  (Configure multisite listeners)
  - バックエンド ターゲットを構成する  
  (Configure back-end targets)
  - リダイレクト ルールを含むルーティング ルールを構成する  
  (Configure routing rules, including redirection rules)
- Azure Traffic Manager プロファイルを実装する  
(Implement an Azure Traffic Manager profile)
  - ルーティング方法 (モード) を構成する  
  (Configure a routing method (mode))
  - エンドポイントを構成する  
  (Configure endpoints)
  - HTTP 設定を作成する  
  (Create HTTP settings)
- Azure 仮想ネットワーク NAT を設計、実装する  
(Design and implement an Azure Virtual Network NAT)
  - 仮想ネットワーク NAT をいつ利用するかを選択する  
  (Choose when to use a Virtual Network NAT)
  - NAT ゲートウェイにパブリック IP やパブリック IP プレフィックスを割り当てる  
  (Allocate public IP or public IP prefixes for a NAT gateway)
  - サブネットに仮想ネットワーク NAT を関連付ける  
  (Associate a Virtual Network NAT with a subnet)
#### ネットワークのセキュア化と監視 (Secure and monitor networks) (15–20%)

(Design, implement, and manage an Azure Firewall deployment)
  - Azure Firewall デプロイメントを設計する

  - Azure Firewall デプロイメントを作成、実装する

  - Azure Virtual WAN ハブ内の Azure Firewall でデプロイされたセキュアなハブを作成する  
  (Create a secure hub by deploying Azure Firewall inside an Azure Virtual WAN hub)
  - サードパーティ NVA に Azure Virtual WAN ハブを統合する

- ネットワーク セキュリティ グループ (NSG) を実装、管理する  
(Implement and manage network security groups (NSGs))

  - アプリケーション セキュリティ グループ (ASG) を作成する

  - ASG を NIC に割り当てる  
  (Associate an ASG to a NIC)
  - NSG ルールを作成、構成する

  - NSG フロー ログを解釈する

  - NSG フロー ルールを検証する  
  (Validate NSG flow rules)

- Web Application Firewall (WAF) デプロイメントを実装する  
(Implement a Web Application Firewall (WAF) deployment)
  - 検出モード、防止モードを構成する  
  (Configure detection or prevention mode)
  - Microsoft マネージドおよびユーザー定義を含む Azure Front Door のルール セットを構成する
