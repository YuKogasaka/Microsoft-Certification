# AZ-204 : Developing Solutions for Microsoft Azure
Microsoft 認定資格「Microsoft Certified : Azure Developer Associate」の対象試験です。
- Exam AZ-204: Developing Solutions for Microsoft Azure - Learn | Microsoft Docs  
https://docs.microsoft.com/learn/certifications/exams/az-204

## 評価されるスキルについて (意訳)
**※2022/04/19 に改定 (アップデート) されました。**
- Study Guide - Exam AZ-204: Developing Solutions for Microsoft Azure
https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE4oZ7B

Azure Developer Associate 認定資格を取得するには、Exam AZ-204: Developing Solutions for Microsoft Azure が必要です。  
(Exam AZ-204: Developing Solutions for Microsoft Azure is required to earn the Azure Developer Associate certification.)

この試験では、次の技術タスクを達成する能力を測定します : Azure コンピューティング ソリューションの開発; Azure ストレージの開発; Azure セキュリティの実装; Azure ソリューションの監視、トラブルシューティング、および最適化; また、Azure サービスとサード パーティ サービスに接続して利用する:試験を開始するときに、問題に含まれるコード言語 (C# または Python) を選択可能です。  
(This exam measures your ability to accomplish the following technical tasks: develop Azure compute solutions; develop for Azure storage; implement Azure security; monitor, troubleshoot, and optimize Azure solutions; and connect to and consume Azure services and third-party services. You’ll be able to select the code language (C# or Python) that’s included in the questions when you launch the exam.)

試験の受験者は、1 ～ 2 年の専門的な開発経験と Microsoft Azure の経験が必要となります。さらに、Azure でサポートされている言語でプログラミングでき、Azure SDK、Azure PowerShell、Azure CLI、データ ストレージ オプション、データ接続、API、アプリの認証と承認、コンピューティングとコンテナーのデプロイ、デバッグ、パフォーマンス チューニング、監視に習熟している必要があります。  
(As an exam candidate, you should have 1–2 years professional development experience and experience with Microsoft Azure. In addition, you should be able to program in a language supported by Azure and proficiency in Azure SDKs, Azure PowerShell, Azure CLI, data storage options, data connections, APIs, app authentication and authorization, compute and container deployment, debugging, performance tuning, and monitoring.)

(Note : Most questions cover features that are general availability (GA). The exam may contain questions on Preview features if those features are commonly used.)

### 機能グループ
#### Azure Compute ソリューションの開発 (Develop Azure compute solutions) (25-30%)
- IaaS ソリューションを実装する  
(Implement IaaS solutions)
  - 仮想マシン (VM) をプロビジョニングする  
  (Provision virtual machines (VMs))
  - ARM テンプレートの構成、検証、およびデプロイする  
  (Configure, validate, and deploy ARM templates)
  - ソリューションのコンテナー イメージを構成する  
  (Configure container images for solutions)
  - Azure Container Registry にイメージを公開する  
  (Publish an image to Azure Container Registry)
  - Azure Container Instance を用いてコンテナーを実行する  
  (Run containers by using Azure Container Instance)
- Azure App Service および Web Apps を作成する  
(Create Azure App Service Web Apps)
  - Azure App Service および Web App を作成する  
  (Create an Azure App Service Web App)
  - 診断ログを有効化する  
  (Enable diagnostics logging)
  - Web アプリをコードにデプロイする  
  (Deploy code to a web app)
  - SSL、API 設定、接続文字列を含む Web アプリ設定を構成する  
  (Configure web app settings including SSL, API settings, and connection strings)
  - スケジュールされた自動スケーリングや運用、もしくはシステム メトリックによる自動スケーリングを含む自動スケーリング ルールを実装する  
  (Implement autoscaling rules including scheduled autoscaling and autoscaling by operational or system metrics)
- Azure Functions を実装する  
(Implement Azure Functions)
  - Azure Functions アプリを作成、デプロイする  
  (Create and deploy Azure Functions apps)
  - 関数の入力、出力バインディングを実装する  
  (Implement input and output bindings for a function)
  - データ操作、タイマー、および Webhook を用いて関数トリガーを実装する  
  (Implement function triggers by using data operations, timers, and webhooks)
  - Azure Durable Functions を実装する  
  (Implement Azure Durable Functions )
#### Azure ストレージの開発 (Develop for Azure storage) (15-20%)
- Cosmos DB ストレージを用いたソリューションを開発する  
(Develop solutions that use Cosmos DB storage)
  - ソリューションに適切な API や SDK を選択する  
  (Select the appropriate API and SDK for a solution)
  - パーティション スキームとパーティション キーを実装する  
  (Implement partitioning schemes and partition keys)
  - データと Cosmos DB コンテナーの操作を実行する  
  (Perform operations on data and Cosmos DB containers)
  - 操作に適切な整合性レベルを設定する  
  (Set the appropriate consistency level for operations)
  - 変更フィード通知を管理する  
  (Manage change feed notifications)
- Blob ストレージを用いたソリューションを開発する  
(Develop solutions that use blob storage)
  - ストレージ アカウント/コンテナー間で Blob ストレージの項目を移動する  
  (Move items in Blob storage between storage accounts or containers)
  - プロパティとメタデータをセット、取得する    
  (Set and retrieve properties and metadata)
  - 適切な SDK を用いてデータに対して操作を実行する  
  (Perform operations on data by using the appropriate SDK)
  - ストレージ ポリシー、データのアーカイブおよび保持期間を実装する  
  (Implement storage policies, data archiving, and retention)
#### Azure セキュリティの実装 (Implement Azure security) (20-25%)
- ユーザー認証および承認を実装する  
(Implement user authentication and authorization)
  - Microsoft ID プラットフォームを用いてユーザーを認証、承認する  
  (Authenticate and authorize users by using the Microsoft Identity platform)
  - Azure Active Directory を用いてユーザーとアプリを認証、承認する  
  (Authenticate and authorize users and apps by using Azure Active Directory)
  - 共有アクセス署名 (SAS) を作成、実装する  
  (Create and implement shared access signatures)
  - Microsoft Graph と相互作用するソリューションを実装する  
  (Implement solutions that interact with Microsoft Graph)
- セキュア クラウド ソリューションを実装する  
(Implement secure cloud solutions)
  - App Configuration と Azure Key Vault を用いてアプリ構成データをセキュアにする  
  (Secure app configuration data by using App Configuration or Azure Key Vault)
  - Azure Key Vault に格納されたキー、シークレット、および証明書を用いてコードを開発する  
  (Develop code that uses keys, secrets, and certificates stored in Azure Key Vault)
  - Azure リソース用のマネージド ID を実装する  
  (Implement Managed Identities for Azure resources)
#### Azure ソリューションの監視、トラブルシュート、最適化 (Monitor, troubleshoot, and optimize Azure solutions) (15-20%)
- ソリューション用のキャッシュを実装する  
(Implement caching for solutions)
  - Azure Cache for Redis のキャッシュおよび有効期限ポリシーを構成する  
  (Configure cache and expiration policies for Azure Cache for Redis)
  - データのサイズ設定、接続、暗号化、有効期限を含む、安全で最適化されたアプリケーション キャッシュ パターンを実装する  
  (Implement secure and optimized application cache patterns including data sizing, connections, encryption, and expiration)
- メトリック データやログ データを用いたソリューションをトラブルシュートする  
(Troubleshoot solutions by using metrics and log data)
  - Application Insights を用いてアプリ/サービスを構成する  
  (• Configure an app or service to use Application Insights)
  - メトリック データやログ データをレビュー、分析する  
  (Review and analyze metrics and log data)
  - Application Insights の Web テストとアラートを実装する  
  (Implement Application Insights web tests and alerts)
#### Azure サービスとサードパーティ製サービスとの接続、消費 (Connect to and consume Azure services and third-party services) (15-20%)
- API Management を実装する  
(Implement API Management)
  - APIM インスタンスを作成する  
  (Create an APIM instance)
  - API を作成、ドキュメント化する  
  (Create and document APIs)
  - API の認証を構成する  
  (Configure authentication for APIs)
  - API のポリシーを定義する  
  (Define policies for APIs)
- イベント ベース ソリューションを開発する  
(Develop event-based solutions)
  - Azure Event Grid を用いてソリューションを実装する  
  (Implement solutions that use Azure Event Grid)
  - Azure Event Hub を用いてソリューションを実装する  
  (Implement solutions that use Azure Event Hub)
- メッセージ ベース ソリューションを開発する  
(Develop message-based solutions)
  - Azure Service Bus を用いてソリューションを実装する  
  (Implement solutions that use Azure Service Bus)
  - Azure Queue Storage キューを用いてソリューションを実装する  
  (Implement solutions that use Azure Queue Storage queues)
