### 試験の対象者
Azure Administrator は、クラウド環境で アイデンティティ、ガバナンス、ストレージ、コンピューティング、および仮想ネットワークを実装、管理、および監視します。また、Azure Administrator は、必要に応じてリソースをプロビジョニング、サイズの設定、監視、および調整します。

この試験の受験者は、Azure の管理に最低 6 か月の実務経験が必要です。また、コア Azure サービス、Azure ワークロード、セキュリティ、およびガバナンスについて十分に理解している必要があります。加えて、PowerShell、コマンドライン インターフェイス、Azure ポータル、および ARM テンプレートの使用経験が必要です。

### スキルの評価
注：評価される各スキルの下に記載しているリストは、スキルの評価項目です。このリストは、決定的でも網羅的でもありません。

注：試験内容について、ほとんどの場合、プレビュー機能はカバーされず、一部の機能は GA (General Availability) となった際に試験に追加されます。

#### Azure ID とガバナンスの管理 (15–20%)
- Azure Active Directory (Azure AD) オブジェクトを管理する
  - ユーザーやグループを作成する
  - 管理ユニットを作成する
  - ユーザーやグループのプロパティを管理する
  - デバイス設定を管理する
  - 一括ユーザー更新を実行する
  - ゲスト アカウントを管理する
  - Azure AD join を構成する
  - セルフサービス パスワード リセットを構成する
- ロールベースのアクセス制御 (RBAC) を管理する
  - カスタム ロールを作成する
  - 異なるスコープで割り当てらえたロールで Azure リソースへのアクセスを提供する
  - アクセス割り当てを解釈する
- サブスクリプションとガバナンスを管理する
  - Azure ポリシーを構成する
  - リソース ロックを構成する
  - リソース タグを適用、管理する
  - リソース グループを管理する
  - サブスクリプションを管理する
  - コストを管理する
  - 管理グループを構成する
#### ストレージの実装と管理 (15–20%)
- ストレージをセキュア化する
  - ストレージ アカウントにネットワーク アクセスを構成する
  - ストレージ アカウントを作成、構成する
  - 共有アクセス署名 (SAS) トークンを生成する
  - アクセス キーを管理する
  - ストレージ アカウントの Azure AD 認証を構成する
  - Azure Files へのアクセスを構成する
- ストレージを管理する
  - Azure ジョブからエクスポートする
  - Azure ジョブにインポートする
  - Azure Storage Explorer をインストール、利用する
  - AZCopy を用いてデータをコピーする
  - Azure Storage レプリケーションを実装する
  - Blob オブジェクト レプリケーションを構成する
- Azure files や Azure Blob ストレージを構成する
  - Azure ファイル共有を作成する
  - Azure File Sync サービスを作成、構成する
  - Azure Blob ストレージを構成する
  - ストレージ層を構成する
  - Blob ライフサイクル管理を構成する
#### Azure Compute リソースのデプロイと管理 (20–25%)
- Azure Resource Manager テンプレートを用いて仮想マシン (VM) のデプロイメントを自動化する
  - Azure Resource Manager テンプレートを修正する
  - 仮想ハード ディスク (VHD) テンプレートを構成する
  - テンプレートからデプロイする
  - Azure Resource Manager テンプレートとしてデプロイメントを保存する
  - 仮想マシン拡張機能をデプロイする
- VM を構成する
  - Azure Disk Encryption を構成する
  - 元のリソースから別のリソースへ VM を移動する
  - VM サイズを管理する
  - データ ディスクを追加する
  - ネットワークを構成する
  - VM を再デプロイする
  - 高可用を構成する
  - スケール セットをデプロイ、構成する
- コンテナーを作成、構成する
  - Azure Container Instance のサイジングやスケーリングを構成する
  - Azure Container Instance のコンテナー グループを構成する
  - Azure Kubernetes Service (AKS) のストレージを構成する
  - AKS のスケーリングを構成する
  - AKS のネットワーク接続を構成する
  - AKS クラスターをアップグレードする
- Azure App Service を作成、構成する
  - App Service プランを作成する
  - App Service プランのスケーリング設定を構成する
  - App Service を作成する
  - App Service をセキュア化する
  - カスタム ドメイン名を構成する
  - App Service のバックアップを構成する
  - ネットワーク設定を構成する
  - デプロイメント設定を構成する
#### 仮想ネットワークの構成と管理 (25–30%)
- 仮想ネットワークを実装、管理する
  - ピアリングを含む仮想ネットワークを作成、構成する
  - プライベート/パブリック IP アドレスを構成する
  - ユーザー定義ネットワーク ルートを構成する
  - サブネットを実装する
  - サブネットにエンドポイントを構成する
  - プライベート エンドポイントを構成する
  - カスタム DNS 設定やプライベート/パブリック DNS ゾーンを含む Azure DNS を構成する
- 仮想ネットワークへのアクセスをセキュア化する
  - セキュリティ ルールを作成する
  - サブネットやネットワーク インターフェースにネットワーク セキュリティグループ (NSG) を関連付ける
  - 効果的なセキュリティ ルールを評価する
  - Azure Firewall を実装する
  - Azure Bastion を実装する
- 負荷分散を構成する
  - Azure Application Gateway を構成する
  - 内部/パブリック負荷分散を構成する
  - 負荷分散をトラブルシュートする
- 仮想ネットワークを監視、トラブルシュートする
  - オンプレミス接続を監視する
  - Azure Monitor for Networks を構成、利用する
  - Azure Network Watcher を利用する
  - 外部ネットワークをトラブルシュートする
  - 仮想ネットワーク接続をトラブルシュートする
- オンプレミス ネットワークを Azure 仮想ネットワークに統合する
  - cAzure VPN Gateway を作成、構成する
  - Azure ExpressRoute を作成、構成する
  - Azure Virtual WAN を構成する
#### Azure リソースの監視とバックアップ (10–15%)
- Azure Monitor を用いてリソースを監視する
  - メトリックを構成、解釈する
  - Azure Monitor ログを構成する
  - ログをクエリ、分析する
  - アラートとアクションをセットアップする
  - Application Insights を構成する
- バックアップと回復を実装する
  - Recovery Services 資格情報コンテナーを作成する
  - バックアップ ポリシーを作成、構成する
  - Azure Backup を用いてバックアップと復元操作を実行する
  - Azure Site Recovery を用いて Site-to-Site 回復を実行する
  - バックアップ レポートを構成、レビューする
