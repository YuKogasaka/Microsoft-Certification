# AZ-500 : Microsoft Azure Security Technologies
Microsoft 認定資格「Microsoft Certified : Azure Security Engineer Associate」の対象試験です。
- Exam AZ-500: Microsoft Azure Security Technologies - Learn | Microsoft Docs  
https://docs.microsoft.com/learn/certifications/exams/az-500

## 試験について (意訳)
**※2022/11/01 に改定 (アップデート) される予定です。**
- Study Guide - Exam AZ-500: Microsoft Azure Security Technologies  
https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE3VC70

この試験の受験者は、エンド ツー エンド インフラストラクチャの一部としてクラウドおよびハイブリッド環境で ID、アクセス、データ、アプリケーション、およびネットワークを保護する Azure セキュリティ コントロールを実装する専門知識を持っている必要があります。  
(Candidates for this exam should have subject matter expertise implementing Azure security controls that protect identity, access, data, applications, and networks in cloud and hybrid environments as part of an end-to-end infrastructure.)

Azure Security Endineer の責任として、セキュリティ体制の管理、脆弱性の特定と修復、脅威モデリングの実行、脅威保護の実装、セキュリティ インシデントのエスカレーションへの対応が含まれます。  
(Responsibilities for an Azure security engineer include managing the security posture, identifying and remediating vulnerabilities, performing threat modeling, implementing threat protection, and responding to security incident escalations.)

多くの場合、Azure セキュリティ エンジニアは、大規模なチームの一員として、クラウドベースの管理とセキュリティを計画および実装します。  
(Azure security engineers often serve as part of a larger team to plan and implement cloud-based management and security.)

また、Azure およびハイブリッド環境の管理に関する実務経験を持っている必要があります。さらに Infrastructure as Code、セキュリティ運用プロセス、クラウド機能、および Azure サービスの経験が必要です。  
(Candidates for this exam should have practical experience in administration of Azure and hybrid environments. Candidates should have experience with infrastructure as code, security operations processes, cloud capabilities, and Azure services.)

### スキルの評価
注：評価される各スキルの下に記載しているリストは、スキルの評価項目です。このリストは、決定的でも網羅的でもありません。

#### ID とアクセスの管理 (Manage identity and access) (30-35%)
- Microsoft Entra の一部である、Azure Active Directory (Azure AD) の ID を管理する  
(Manage identities in Microsoft Azure Active Directory (Azure AD), part of Microsoft Entra)
  - Azure リソースのマネージド ID を作成、管理する  
  (Create and manage a managed identity for Azure resources)
  - Azure AD グループを管理する  
  (Manage Azure AD groups)
  - Azure AD ユーザーを管理する  
  (Manage Azure AD users)
  - Azure AD を用いて外部 ID を管理する  
  (Manage external identities by using Azure AD)
  - 管理単位を管理する  
  (Manage administrative units)
-Microsoft Entra の一部である、Azure Active Directory (Azure AD) を用いてセキュアなアクセスを管理する  
(Manage secure access by using Microsoft Azure Active Directory (Azure AD), part of Microsoft Entra)
  - Azure AD Privileged Identity Management (PIM) を構成する  
  (Configure Azure AD Privileged Identity Management (PIM))
  - Multi-Factor Authentication (MFA) を含む条件付きアクセス ポリシーを実装する  
  (Implement Conditional Access policies, including multifactor authentication)
  - Azure AD ID 保護を実装する  
  (Implement Azure AD Identity Protection)
  - パスワードレス認証を実装する  
  (Implement passwordless authentication)
  - アクセス レビューを構成する  
  (Configure access reviews)
- アプリケーションのアクセスを管理する  
(Manage application access)
  - 認証に対するシングル サインオン (SSO) および ID プロバイダーを統合する  
  (Integrate single sign-on (SSO) and identity providers for authentication)
  - App Registration を作成する  
  (Create an app registration)
  - App Registration のアクセス許可スコープを構成する  
  (Configure app registration permission scopes)
  - App Registration のアクセス許可の同意を管理する  
  (Manage app registration permission consent)
  - Azure サブスクリプションやリソースへ API アクセス許可を管理する  
  (Manage API permissions to Azure subscriptions and resources)
  - サービス プリンシパルの認証メソッドを構成する  
  (Configure an authentication method for a service principal)
- アクセス制御を管理する  
(Manage access control)
  - 管理グループ、サブスクリプション、リソース グループ、およびリソースに対する Azure ロールのアクセス許可を構成する  
  (Configure Azure role permissions for management groups, subscriptions, resource groups, and resources)
  - ロールとリソースのアクセス許可を解釈する  
  (Interpret role and resource permissions)
  - Microsoft Entra の一部である、Microsoft Azure Active Directory (Azure AD) のビルトイン ロールを割り当てる  
  (Assign built-in roles in Microsoft Azure Active Directory (Azure AD), part of Microsoft Entra)
  - Azure ロールおよび Azure AD ロールを含むカスタム ロールを作成、割り当てる  
  (Create and assign custom roles, including Azure roles and Azure AD roles)
#### プラットフォーム保護の実装 (Implement platform protection) (15-20%)
- 高度なネットワーク セキュリティを実装する  
(Implement advanced network security)
  - ハイブリッド ネットワークの接続をセキュアにする  
  (Secure the connectivity of hybrid networks)
  - 仮想ネットワークの接続をセキュアにする  
  (Secure the connectivity of virtual networks)
  - Azure Firewall を作成、構成する  
  (Create and configure Azure Firewall)
  - Azure Firewall Manager を作成、構成する  
  (Create and configure Azure Firewall Manager)
  - Azure Application Gateway 作成、構成する  
  (Create and configure Azure Application Gateway)
  - Azure Front Door を作成、構成する  
  (Create and configure Azure Front Door)
  - Web Application Firewall (WAF) を作成、構成する  
  (Create and configure Web Application Firewall (WAF))
  - ストレージ アカウント、Azure SQL、Azure Key Vault、Azure App Service を含むリソース ファイアウォールを構成する  
  (Configure a resource firewall, including storage account, Azure SQL, Azure Key Vault, or Azure App Service)
  - Web Apps および Azure Functions のネットワーク分離を構成する  
  (Configure network isolation for Web Apps and Azure Functions)
  - Azure サービス エンドポイントを実装する  
  (Implement Azure Service Endpoints)
  - 他のサービスとの統合を含む Azure プライベート エンドポイントを実装する  
  (Implement Azure Private Endpoints, including integrating with other services)
  - Azure プライベート リンクを実装する  
  (Implement Azure Private Links)
  - Azure DDoS 保護を実装する  
  (Implement Azure DDoS Protection)
- Compute の高度なセキュリティを構成する  
(Configure advanced security for compute)
  - 仮想マシン (VM) のエンドポイント保護を構成する  
  (Configure Endpoint Protection for virtual machines (VMs))
  - VM のセキュリティ アップデートを実装、管理する  
  (Implement and manage security updates for VMs)
  - コンテナー サービスのセキュリティを構成する  
  (Configure security for container services)
  - Azure Container Registry へのアクセスを管理する  
  (Manage access to Azure Container Registry)
  - サーバーレス Compute のセキュリティを構成する  
  (Configure security for serverless compute)
  - Azure App Service のセキュリティを構成する  
  (Configure security for an Azure App Service)
  - Rest の暗号化を構成する  
  (Configure encryption at rest)
  - トランジットの暗号化を構成する  
  (Configure encryption in transit)
#### セキュリティ操作の管理 (Manage security operations) (25-30%)
- 集中化されたポリシー管理を構成する  
(Configure centralized policy management)
  - カスタム セキュリティ ポリシーを構成する  
  (Configure a custom security policy)
  - ポリシー イニシアチブを作成する  
  (Create a policy initiative)
  - Azure Policy を用いてセキュリティ設定および監査を構成する  
  (Configure security settings and auditing by using Azure Policy)
- 脅威の保護を構成、管理する  
(Configure and manage threat protection)
  - (Microsoft Defender for Endpoint を含まない) Microsoft Defender for Servers を構成する  
  (Configure Microsoft Defender for Servers (not including Microsoft Defender for Endpoint))
  - Microsoft Defender for Servers から脆弱性スキャンを評価する  
  (Evaluate vulnerability scans from Microsoft Defender for Cloud)
  - Microsoft Defender for SQL を構成する  
  (Configure Microsoft Defender for SQL)
  - Microsoft Threat Modeling Tool を利用する  
  (Use the Microsoft Threat Modeling Tool)
- セキュリティ監視ソリューションを構成、管理する  
(Configure and manage security monitoring solutions)
  - Azure Monitor を用いてアラート ルールを作成、カスタマイズする  
  (Create and customize alert rules by using Azure Monitor)
  - Azure Monitor を用いて診断ログおよびログの保持期間を構成する  
  (Configure diagnostic logging and log retention by using Azure Monitor)
  - Azure Monitor を用いてセキュリティ ログを監視する  
  (Monitor security logs by using Azure Monitor)
  - Microsoft Sentinel のアラート ルールを作成、カスタマイズする  
  (Create and customize alert rules in Microsoft Sentinel)
  - Microsoft Sentinel のデータ接続を構成する  
  (Configure data connectors in Microsoft Sentinel)
  - Microsoft Sentinel のアラートおよびインシデントを評価する  
  (Evaluate alerts and incidents in Microsoft Sentinel)
#### データとアプリケーションのセキュア化 (Secure data and applications) (25-30%)
- ストレージのセキュリティを構成する  
(Configure security for storage)
  - ストレージ アカウントのアクセス制御を構成する  
  (Configure access control for storage accounts)
  - ストレージ アカウント アクセス キーを構成する  
  (Configure storage account access keys)
  - SMB アクセスに対する Azure Files の ID ベース認証を構成する  
  (Configure Azure Files identity-based authentication for SMB access)
  - 委任されたアクセスを構成する  
  (Configure delegated access)
- データ セキュリティを構成する  
(Configure security for data)
  - Microsoft Entra の一部である、Microsoft Azure Active Directory (Azure AD) を用いてデータベース認証を有効化する  
  (Enable database authentication by using Microsoft Azure Active Directory (Azure AD), part of Microsoft Entra)
  - データベース監査を有効化する  
  (Enable database auditing)
  - SQL ワークロードの動的マスクを構成する  
  (Configure dynamic masking on SQL workloads)
  - Azure SQL データベースのデータベース暗号化を実装する  
  (Implement database encryption for Azure SQL Database)
  - Azure Synapse Analytics および Azure Cosmos DB を含むデータ ソリューションのネットワーク分離を実装する  
  (Implement network isolation for data solutions, including Azure Synapse Analytics and Azure Cosmos DB)
- Azure Key Vault を構成、管理する  
(Configure and manage Azure Key Vault)
  - Key Vault を作成、構成する  
  (Create and configure Key Vault)
  - Key Vault へのアクセスを構成する  
  (Configure access to Key Vault)
  - 証明書、シークレット、およびキーを管理する  
  (Manage certificates, secrets, and keys)
  - キー ローテーションを構成する  
  (Configure key rotation)
  - 証明書、シークレット、およびキーをバックアップ、復元する  
  (Configure backup and recovery of certificates, secrets, and keys)
