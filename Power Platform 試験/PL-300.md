## 評価されるスキルについて (意訳)
**※2023/01/31 に英語版が改定 (アップデート) されました。**
- Exam PL-300: Microsoft Power BI Data Analyst – Skills Measured  
https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RWREll

受験者は、利用可能なデータを操作し、ドメインの専門知識を適用することにより、実用的なインサイトを提供します。理解しやすいデータの視覚化を通じて有意義なビジネス価値を提供し、他のユーザーがセルフサービス分析を実行できるようにし、使用するソリューションをデプロイ、構成できるようにします。
(Candidates for this exam deliver actionable insights by working with available data and applying domain expertise. They provide meaningful business value through easy-to-comprehend data visualizations, enable others to perform self-service analytics, and deploy and configure solutions for consumption.)

Power BI データ アナリストは、ビジネスの利害関係者と緊密に連携して、ビジネス要件を特定します。企業のデータ アナリストやデータ エンジニアと協力して、データを特定して取得します。また、Power BI を用いて、データの変換、データ モデルの作成、データの視覚化、資産の共有も行います。
(The Power BI data analyst works closely with business stakeholders to identify business requirements. They collaborate with enterprise data analysts and data engineers to identify and acquire data. They also transform the data, create data models, visualize data, and share assets by using Power BI.)

さらに受験者は、Power Query の利用と、Data Analysis Expressions (DAX) を使用した式の作成に習熟している必要があります。加えて、データ品質を評価する方法を知っています。さらに、行レベルのセキュリティやデータの機密性など、データ セキュリティについても理解しています。
(Candidates for this exam should be proficient at using Power Query and writing expressions by using Data Analysis Expressions (DAX). These professionals know how to assess data quality. Plus, they understand data security, including row-level security and data sensitivity.)

### 機能グループ
#### データの準備 (Prepare the data) (25–30%)
- データ ソースからデータを取得する  
  (Get data from data sources)
  - データ ソースを特定し、接続する
    (Identify and connect to a data source)
  - 資格情報、プライバシー レベル、データ ソースの場所を含む、データ ソースの設定を変更する  
    (Change data source settings, including credentials, privacy levels, and data source locations)
  - 共有データセットを選択するか、ローカル データセットを作成する
    (Select a shared dataset, or create a local dataset)
  - DirectQuery、インポート、およびデュアル モードから選択する  
    (Choose between DirectQuery, Import, and Dual mode)
  - パラメータ値を変更する
    (Change the value in a parameter)
- データをきれいにする  
  (Clean the data)
  - データ統計や列のプロパティを含む、データを評価する  
    (Evaluate data, including data statistics and column properties)
  - 不整合、予期しない値または null 値、およびデータ品質の問題を解決する  
    (Resolve inconsistencies, unexpected or null values, and data quality issues)
  - データのインポート エラーを解決する  
    (Resolve data import errors)
- データを変換してロードする  
  (Transform and load the data)
  - 適切な列のデータ型を選択する  
    (Select appropriate column data types)
  - 列を作成、変換する  
    (Create and transform columns)
  - クエリを変換する  
    (Transform a query)
  - ファクトとディメンションを含むスター スキーマを設計する  
    (Design a star schema that contains facts and dimensions)
  - 参照クエリ、もしくは重複クエリを利用するタイミングとその影響を特定する  
    (Identify when to use reference or duplicate queries and the resulting impact)
  - クエリをマージ、追加する  
    (Merge and append queries)
  - 関係の適切なキーを特定し、作成する  
    (Identify and create appropriate keys for relationships)
  - クエリのデータ読み込みを構成する  
    (Configure data loading for queries)
#### データのモデル化 (Model the data) (25–30%)
- データ モデルを設計、実装する  
  (Design and implement a data model)
  - テーブルと列のプロパティを構成する  
    (Configure table and column properties)
  - ロールプレイング ディメンションを実装する  
    (Implement role-playing dimensions)
  - リレーションシップのカーディナリティとクロスフィルターの方向を定義する  
    (Define a relationship's cardinality and cross-filter direction)
  - 共通の日付テーブルを作成する  
    (Create a common date table)
  - 行レベルのセキュリティ ロールを実装する  
    (Implement row-level security roles)
- DAX を使用してモデル計算を作成する  
  (Create model calculations by using DAX)
  - 単一の集計メジャーを作成する  
    (Create single aggregation measures)
  - CALCULATE を用いてフィルターを操作する  
    (Use CALCULATE to manipulate filters)
  - タイム インテリジェンス対策を実装する  
    (Implement time intelligence measures)
  - 暗黙の測定値を特定し、明示的な測定値に置き換える  
    (Identify implicit measures and replace with explicit measures)
  - 基本的な統計関数を利用する  
    (Use basic statistical functions)
  - 準加法メジャーを作成する  
    (Create semi-additive measures)
  - クイック メジャーを用いてメジャーを作成する  
    (Create a measure by using quick measures)
  - 計算テーブルを作成する  
    (Create calculated tables)
- モデルのパフォーマンスを最適化する  
  (Optimize model performance)
  - 不要な行と列を特定して削除することにより、パフォーマンスを向上させる  
    (Improve performance by identifying and removing unnecessary rows and columns)
  - パフォーマンス アナライザーを用いて、パフォーマンスの低いメジャー、関係、およびビジュアルを特定する  
    (Identify poorly performing measures, relationships, and visuals by using Performance Analyzer)
  - 最適なデータ型を選択し、パフォーマンスを向上させる  
    (Improve performance by choosing optimal data types)
  - データを要約し、パフォーマンスを向上させる  
    (Improve performance by summarizing data)
#### データの可視化と分析 (Visualize and analyze the data) (25—30%)
- レポートを作成する  
  (Create reports)
  - 適切な視覚化を特定、実装する  
    (Identify and implement appropriate visualizations)
  - ビジュアライゼーションをフォーマット、構成する  
    (Format and configure visualizations)
  - カスタム ビジュアルを利用する  
    (Use a custom visual)
  - テーマを適用してカスタマイズする  
    (Apply and customize a theme)
  - 条件付き書式を構成する  
    (Configure conditional formatting)
  - スライスとフィルタリングを適用する  
    (Apply slicing and filtering)
  - レポート ページを構成する  
    (Configure the report page)
  - Excel で分析機能を利用する  
    (Use the Analyze in Excel feature)
  - ページ分割されたレポートをいつ利用するかを選択する  
    (Choose when to use a paginated report)
- 使いやすさとストーリーテリングのためにレポートを強化する  
  (Enhance reports for usability and storytelling)
  - ブックマークを構成する  
    (Configure bookmarks)
  - カスタム ツールチップを作成する  
    (Create custom tooltips)
  - ビジュアル間の相互作用を編集、および構成する  
    (Edit and configure interactions between visuals)
  - レポートのナビゲーションを構成する  
    (Configure navigation for a report)
  - 並べ替えを適用する  
    (Apply sorting)
  - 同期スライサーを構成する  
    (Configure sync slicers)
  - 選択ウィンドウを使用してビジュアルをグループ化およびレイヤー化する  
    (Group and layer visuals by using the Selection pane)
  - インタラクティブなビジュアルを用いてデータにドリルダウンする  
    (Drill down into data using interactive visuals)
  - レポート コンテンツのエクスポートを構成し、エクスポートを実行する  
    (Configure export of report content, and perform an export)
  - モバイル デバイス用のレポートを設計する  
    (Design reports for mobile devices)
  - レポートに Q&A 機能を組み込む  
    (Incorporate the Q&A feature in a report)
- パターンと傾向を特定する  
  (Identify patterns and trends)
  - Power BI で分析機能を利用する  
    (Use the Analyze feature in Power BI)
  - グループ化、ビニング、およびクラスタリングを利用する  
    (Use grouping, binning, and clustering)
  - AI ビジュアルを利用する  
    (Use AI visuals)
  - 基準線、誤差範囲、予測を使用する  
    (Use reference lines, error bars, and forecasting)
  - 外れ値と異常を検出する  
    (Detect outliers and anomalies)
  - スコアカードと指標を作成、共有する  
    (Create and share scorecards and metrics)
#### アセットの展開と保守 (Deploy and maintain assets) (15–20%)
- ワークスペースとアセットを作成、管理する  
  (Create and manage workspaces and assets)
  - ワークスペースを作成、構成する  
    (Create and configure a workspace)
  - ワークスペースの役割を割り当てる  
    (Assign workspace roles)
  - ワークスペース アプリを構成、アップデートする  
    (Configure and update a workspace app)
  - ワークスペースでアセットを公開、インポート、もしくはアップデートする  
    (Publish, import, or update assets in a workspace)
  - ダッシュボードを作成する  
    (Create dashboards)
  - 配布方法を選択する  
    (Choose a distribution method)
  - ワークスペースのコンテンツに機密ラベルを適用する  
    (Apply sensitivity labels to workspace content)
  - サブスクリプションとデータ アラートを構成する  
    (Configure subscriptions and data alerts)
  - Power BI コンテンツを宣伝、認定する  
    (Promote or certify Power BI content)
  - ファイルのグローバル オプションを管理する  
    (Manage global options for files)
- データセットを管理する  
  (Manage datasets)
  - ゲートウェイが必要な場合を特定する  
    (Identify when a gateway is required)
  - データセットのスケジュールされた更新を構成する  
    (Configure a dataset scheduled refresh)
  - 行レベルのセキュリティ グループ メンバーシップを構成する  
    (Configure row-level security group membership)
  - データセットへのアクセスを提供する  
    (Provide access to datasets)
