# PL-900 : Microsoft Power Platform Fundamentals
Microsoft 認定資格「Microsoft Certified : Power Platform Fundamentals」の対象試験です。
- Exam PL-900: Microsoft Power Platform Fundamentals - Learn | Microsoft Docs  
https://docs.microsoft.com/learn/certifications/exams/pl-900

## 試験について (意訳)
**※2023/03/24 に英語版が改定 (アップデート) される予定です。**
- Study Guide - Exam PL-900: Microsoft Power Platform Fundamentals  
https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE43Nnn

試験対象者は、Microsoft Power Platform テクノロジを用いてプロセスを改善し、ビジネスの成果を促進する方法を理解することを目指しています。この試験は、Microsoft Power Platform を使用してソリューションを構築するジャーニーを始めようとしている受験者を対象としています。  
(Candidates for this exam aspire to understand how Microsoft Power Platform technologies can be used to improve processes and help drive business outcomes. The exam is intended for candidates seeking to start their journey building solutions with Microsoft Power Platform.)

この試験は、ロール ベース、もしくは専門の認定資格の準備に使用できますが、いずれの前提条件でもありません。  
(This exam can be used to prepare for role-based or specialty certifications, but it is not a prerequisite for any of them.)

### 対象となる領域 : 試験で評価されるスキル
試験は定期的にアップデートされ、役割を果たすために必要なスキルが反映されます。試験を受ける時期に応じて、スキル測定目標の 2 つのバージョンが含まれています。  
(Our exams are updated periodically to reflect skills that are required to perform a role. We have included two versions of the Skills Measured objectives depending on when you are taking the exam.)

常に、試験の英語版を最初にアップデートします。一部の試験は他の言語にローカライズされており、英語版がアップデートされてから約 8 週間後にアップデートされます。その他の利用可能な言語は、試験の詳細 Web ページの試験のスケジュールセクションに一覧表示されています。ご希望の言語で試験を利用できない場合は、試験を完了するために 30 分の追加時間をリクエストすることが可能です。  
(We always update the English language version of the exam first. Some exams are localized into other languages, and those are updated approximately eight weeks after the English version is updated. Other available languages are listed in the Schedule Exam section of the Exam Details webpage. If the exam isn't available in your preferred language, you can request an additional 30 minutes to complete the exam.)

注 : 評価される各スキル続いて記載している箇条書きの一覧は、スキルの評価項目 (どのように評価しているかを示す) となります。関連するトピックが試験内でカバーされている場合があります。  
(Note : The bullets that follow each of the skills measured are intended to illustrate how we are assessing that skill. Related topics may be covered in the exam.)

注 : ほとんどの問題は、GA (General Availability) となった機能に関するものです。プレビュー機能が一般的に使用されている場合、それらに関する問題が含まれる場合があります。  
(Note : Most questions cover features that are general availability (GA). The exam may contain questions on Preview features if those features are commonly used.)

### 機能グループ
#### Microsoft Power Platform のビジネス価値を説明する (Describe the business value of Microsoft Power Platform) (20–25%)
- Microsoft Power Platform サービスのビジネス価値を説明する  
  (Describe the business value of Microsoft Power Platform services)
  - アプリケーションをビルドするための Power Apps の価値を説明する  
    (Describe the value of Power Apps to build applications)
  - プロセスを自動化するための Power Automate の価値を説明する  
    (Describe the value of Power Automate to automate processes)
  - データへのインサイトをを得るための Power BI の価値を説明する  
    (Describe the value of Power BI to gain insights into data)
  - Web サイトを構築するための Power Pages の価値を説明する  
    (Describe the value of Power Pages to build websites)
  - チャットボットを作成するための Power Virtual Agents の価値を説明する  
    (Describe the value of Power Virtual Agents to create chatbots)
  - サービスとデータを統合するためのコネクタの価値について説明する  
    (Describe the value of connectors to integrate services and data)
  - ビジネス データを整理するための Microsoft Dataverse の価値を説明する  
    (Describe the value of Microsoft Dataverse to organize business data)
  - アプリとフローを強化するための AI Builder の価値を説明する  
    (Describe the value of AI Builder to enhance apps and flows)
- Microsoft Power Platform を用いてビジネス ソリューションを拡張するビジネス価値を説明する  
  (Describe the business value of extending business solutions by using Microsoft Power Platform)
  - Microsoft Power Platform アプリが Microsoft Dynamics 365 アプリとどのように連携するかを説明する  
    (Describe how Microsoft Power Platform apps work together with Microsoft Dynamics 365 apps)
  - Microsoft Power Platform ビジネス ソリューションが Microsoft 365 のアプリやサービスとどのように連携するかを説明する  
    (Describe how Microsoft Power Platform business solutions work together with Microsoft 365 apps and services)
  - どのように Microsoft Power Platform アプリがともに動作するかを説明する  
    (Describe how Microsoft Power Platform apps work together)
  - Microsoft Power Platform が Microsoft Teams とどのように連携するかを説明する  
    (Describe how Microsoft Power Platform works with Microsoft Teams)
  - Microsoft Power Platform ビジネス ソリューションが Microsoft Azure サービスをどのように利用できるかを説明する  
    (Describe how Microsoft Power Platform business solutions can consume Microsoft Azure services)
- Microsoft Power Platform の管理とガバナンスについて説明する
  (Describe Microsoft Power Platform administration and governance)
  - Microsoft Power Platform のセキュリティ モデルについて説明する  
    (Describe the Microsoft Power Platform security model)
  - 環境を説明する  
    (Describe environments)
  - Microsoft Power Platform 管理センターおよび Microsoft 365 管理センターを含む特定の管理タスクを実行する場所を説明する  
    (Describe where to perform specific administrative tasks including Microsoft Power Platform admin center and Microsoft 365 admin center)
  - Microsoft Power Platform がプライバシーとアクセシビリティのガイドラインをどのようにサポートするかを説明する  
    (Describe how Microsoft Power Platform supports privacy and accessibility guidelines)
#### Microsoft Power Platform の基本的なコンポーネントを認識する (Identify foundational components of Microsoft Power Platform) (10–15%)
- Microsoft Dataverse を説明する  
  (Describe Microsoft Dataverse)
  - 従来のデータベースと Microsoft Dataverse の違いについて説明する  
    (Describe differences between traditional databases and Microsoft Dataverse)
  - Microsoft Dataverse のテーブル、列、リレーションシップについて説明する  
    (Describe tables, columns, and relationships in Microsoft Dataverse)
  - Microsoft Dataverse でビジネス ロジックを記述する  
    (Describe business logic in Microsoft Dataverse)
  - Microsoft Dataverse のデータフローについて説明する  
    (Describe dataflows for Microsoft Dataverse)
- コネクターを説明する  
  (Describe connectors)
  - コネクターのコンポーネントについて説明する  
    (Describe the components of a connector)
  - 標準コネクターおよびプレミアム コネクターについて説明する  
    (Describe standard and premium connectors)
  - カスタム コネクタのユース ケースを説明する  
    (Identify use cases for custom connectors)
#### Power BI の機能をデモンストレーションする (Demonstrate the capabilities of Power BI) (10–15%)
- 基本的な Power BI コンポーネントを特定する  
  (Identify basic Power BI components)
  - Power BI Desktop および Power BI サービスについて説明する  
    (Describe Power BI Desktop and Power BI Service)
  - ダッシュボード、ワークスペース、レポート、およびページ分割されたレポートを比較、対照する  
    (Compare and contrast dashboards, workspaces, reports, and paginated reports)
  - Power Query を用いてデータをクリーニングおよび変換する方法について説明する  
    (Describe how to clean and transform data by using Power Query)
  - AI Insights がデータの異常と傾向の特定にどのように役立つかを説明する  
    (Describe how AI Insights helps identify anomalies and trends in data)
- Power BI を用いて基本的なダッシュボードをビルドする  
  (Build a basic dashboard by using Power BI)
  - 視覚エフェクトを追加し、Power BI レポートを作成する  
    (Create a Power BI report by adding visualizations)
  - Power BI ダッシュボードを作成する  
    (Create a Power BI dashboard)
  - Add visualizations to dashboards by using Q&A
  - Consume Power BI reports and dashboards

#### Power Automate の機能をデモンストレーションする (Demonstrate the capabilities of Power Automate) (15–20%)
- 基本的な Power Automate コンポーネントを特定する  
  (Identify basic Power Automate components)
  - クラウドおよびデスクトップ フローの使用例を説明する  
    (Describe use cases for cloud and desktop flows)
  - クラウド フロー テンプレートの使用例を説明する  
    (Describe use cases for cloud flow templates)
  - どのように Power Automate クラウド フローがコネクタ トリガーとアクションを使用する方法を説明する  
    (Describe how Power Automate cloud flows use connector triggers and actions)
  - Microsoft Teams、Outlook、SharePoint、Forms の承認に対する Power Automate ユース ケースについて説明する  
    (Describe Power Automate use cases for approvals, Microsoft Teams, Outlook, SharePoint, and Forms)
  - Power Automate for Desktop、Power Automate モバイル、Power Automate ポータルなどの Power Automate アプリについて説明する  
    (Describe the Power Automate apps including Power Automate for Desktop, Power Automate mobile, and the Power Automate portal)
  - Power Automate デスクトップ フローのアクションについて説明する  
    (Describe actions for Power Automate desktop flows)
  - Power Automate でのドキュメント処理について説明する  
    (Describe document processing in Power Automate)
- Build a basic Power Automate flow
  - インスタント、自動化、もしくはスケジュールされたフロー テンプレートを用いてクラウド フローを作成する

  - クラウド フローを修正する
