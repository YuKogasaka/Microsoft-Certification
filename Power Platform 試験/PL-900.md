### 機能グループ
#### Microsoft Power Platform のビジネス価値を説明する (Describe the business value of Microsoft Power Platform) (20-25%)
- Microsoft Power Platform サービスのビジネス価値を説明する  
  (Describe the business value of Microsoft Power Platform services)
  - Power BI を用いてデータへのインサイトを取得する  
    (Gain insights into data by using Power BI)
  - Power Apps を用いてアプリケーションをすばやくビルドする  
    (Build applications quickly by using Power Apps)
  - Power Automate を用いて処理を自動化する  
    (Automate processes by using Power Automate)
  - サービスやデータにアクセスするためのコネクタを使用する  
    (Use connectors to access services and data)
  - Power Virtual Agents Web アプリや Microsoft Teams の Power Virtual Agents を用いて、強力なチャットボットを作成する  
    (Create powerful chatbots by using the Power Virtual Agents web app and Power Virtual Agents in Microsoft Teams)
- Microsoft Power Platform を用いてビジネス ソリューションを拡張するビジネスの価値を説明する  
    (Describe the business value of extending business solutions by using Microsoft Power Platform)
  - Dynamics 365 アプリとともにどのように Microsoft Power Platform アプリを動作させるかを説明する  
    (Describe how Microsoft Power Platform apps work together with Dynamics 365 apps)
  - Microsoft Power Platform ビジネス ソリューションを Microsoft 365 アプリやサービスを用いる方法を説明する  
    (Describe how Microsoft Power Platform business solutions can be used by Microsoft 365 apps and services)
  - Microsoft Teams を用いて Microsoft Power Platform ソリューションを用いるかどうかを説明する  
    (Describe how to use Microsoft Power Platform solutions with Microsoft Teams)
  - Microsoft Power Platform ビジネス ソリューションが Azure Cognitive Services を含む Microsoft Azure サービスをどのように利用できるかを説明する  
    (Describe how Microsoft Power Platform business solutions can consume Microsoft Azure services including Azure Cognitive Services)
  - Microsoft Power Platform ビジネス ソリューションがサードパーティのアプリやサービスをどのように利用できるかを説明する  
    (Describe how Microsoft Power Platform business solutions can consume third-party apps and services)
  - AppSource のユース ケースを説明する  
    (Describe use cases for AppSource)
  - どのように Microsoft Power Platform アプリがともに動作するかを説明する  
    (Describe how Microsoft Power Platform apps work together)
- Microsoft Power Platform の管理とセキュリティを説明する  
  (Describe Microsoft Power Platform administration and security)
  - Microsoft Dataverse セキュリティ ロール、Azure Identity Services、Access Management (IAM) の認識を含む Microsoft Power Platform がセキュリティを実装する方法を説明する  
    (Describe how Microsoft Power Platform implements security including awareness of Microsoft Dataverse security roles, Azure Identity Services, and Access Management (IAM))
  - アプリやユーザーをどのように管理するかを説明する  
    (Describe how to manage apps and users)
  - 環境を説明する  
    (Describe environments)
  - Power Platform 管理センターおよび Microsoft 365 管理センターを含む特定の管理タスクを実行する場所を説明する  
    (Describe where to perform specific administrative tasks including Microsoft Power Platform Admin center and Microsoft 365 admin center)
  - データ ポリシーを説明する  
    (Describe data policies)
  - Microsoft Power Platform がプライバシーとアクセシビリティのガイドラインをどのようにサポートするかを説明する  
    (Describe how Microsoft Power Platform supports privacy and accessibility guidelines)
  - Microsoft Power Platform プライバシーとアクセシビリティ機能を説明する  
    (Describe Microsoft Power Platform privacy and accessibility capabilities)
  - Microsoft Power Platform ガバナンス機能を説明する  
    (Describe Microsoft Power Platform governance capabilities)
#### Microsoft Power Platform のコア コンポーネントを特定する (Identify the core components of Microsoft Power Platform) (10-15%)
- Microsoft Dataverse を説明する  
  (Describe Microsoft Dataverse)
  - データベースと Dataverse との違いを説明する  
    (Describe the difference between databases and Dataverse)
  - Dataverse と Dataverse for Teams との違いを説明する  
    (Describe the differences between Dataverse and Dataverse for Teams)
  - テーブル、列、およびリレーションシップを説明する  
    (Describe tables, columns, and relationships)
  - 人、場所、物事を説明する際に一般的な標準テーブルをどのように用いるかを説明する  
    (Describe how to use common standard tables to describe people, places, and things)
- コネクタを説明する  
  (Describe Connectors)
  - トリガー タイプを含むトリガーとトリガーが使用される場所を説明する  
    (Describe triggers including trigger types and where triggers are used)
  - アクションを説明する  
    (Describe actions)
  - standard/premium 層を含むコネクタのライセンス オプションを説明する  
    (Describe licensing options for connectors including standard or premium tier )
  - カスタム コネクタのユース ケースを説明する  
    (Identify use cases for custom connectors)
- AI Builder を説明する  
  (Describe AI Builder)
  - AI Builder のビジネス価値を特定する  
    (Identify the business value of AI Builder)
  - 名刺リーダー、検出モデル、フォーム処理モデル、予測モデルを含むモデルを説明する  
    (Describe models including business card reader, detection model, form processing model, and prediction model)
  - Power Apps や Power Automate が AI Builder データをどのように消費するかを説明する  
    (Describe how the Power Apps and Power Automate can consume AI Builder data)
#### Power BI の機能をデモンストレーションする (Demonstrate the capabilities of Power BI) (20-25%)
- 一般的な Power BI コンポーネントを特定する  
  (Identify common Power BI components)
  - 円グラフ、棒グラフ、ドーナツ グラフ、散布図、KPI を含む視覚化コントロールの使用方法を特定、説明する  
    (Identify and describe uses for visualization controls including pie, bar, donut, and scatter plots and KPIs)
  - Power BI Desktop Reports、Data、および Model ビューを説明する  
    (Describe the Power BI Desktop Reports, Data, and Model views)
  - Power BI Desktop と Power BI Service を比較、対比する  
    (Compare and contrast Power BI Desktop and Power BI Service)
  - ダッシュボード、ワークスペース、レポートを比較、対比する  
    (Compare and contrast dashboards, workspaces, and reports)
  - Power BI セキュリティ モデルを説明する  
    (Describe the Power BI security model)
  - モバイル アプリでの Power BI の利用について説明する  
    (Describe using Power BI in mobile apps)
- データを接続、消費する  
  (Connect to and consume data)
  - Microsoft Excel を含む複数のソースからのデータに接続して結合する  
    (Connect to and combine data from multiple sources including Microsoft Excel)
  - Power Query を用いてデータをクリーンアップおよび変換する方法を説明する  
    (Describe how to use Power Query to clean and transform data)
  - 集計関数を説明、実装する  
    (Describe and implement aggregate functions)
  - Microsoft Excel を含む利用可能なデータ ソースの種類を特定する  
    (Identify available types of data sources including Microsoft Excel)
  - 共有データセットのユース ケースを説明する  
    (Describe use cases for shared datasets)
  - テンプレート アプリのユース ケースを説明する  
    (Describe use cases for template apps)
  - Power BI レポートとダッシュボードを表示するためのオプションを説明する  
    (Describe options for viewing Power BI reports and dashboards)
- Power BI を用いて基本的なダッシュボードをビルドする  
  (Build a basic dashboard using Power BI)
  - Power BI レポートを作成する  
    (Create a Power BI report)
  - Power BI ダッシュボードを作成する  
    (Create a Power BI dashboard)
  - レポートとダッシュボードを公開、共有する  
    Publish and share reports and dashboards)
#### Power Apps の機能をデモンストレーションする (Demonstrate the capabilities of Power Apps) (25-30%)
- 一般的な Power Apps コンポーネントを特定する  
  (Identify common Power Apps components)
  - キャンバス アプリとモデル駆動アプリの違いを説明する  
     (Describe the differences between canvas apps and model-driven apps)
  - 数式のユース ケースを説明する  
    (Describe use cases for formulas)
- 基本的なキャンバス アプリをビルドする  
  (Build a basic canvas app)
  - データ ソースのタイプを説明する  
    (Describe types of data sources)
  - コネクタを用いてデータを接続する  
    (Connect to data by using connectors)
  - データからアプリを作成する  
    (Create an app from data)
  - ユーザー エクスペリエンスを設計するコントロールを使用する  
    (Use controls to design the user experience)
  - アプリを公開、共有する  
    (Publish and share an app)
- 基本的なモデル駆動アプリをビルドする  
  (Build a basic model-driven app)
  - テーブルからモデル駆動アプリを作成する  
    (Create a model-driven app from tables)
  - フォームを修正する  
    (Modify forms)
  - ビューを作成、修正する  
    (Create and modify views)
  - アプリを公開、共有する  
    (Publish and share an app)
#### Power Automate の機能をデモンストレーションする (Demonstrate the capabilities of Power Automate) (10-15%)
- 一般的な Power Automate コンポーネントを特定する  
  (Identify common Power Automate components)
  - クラウド、デスクトップ、およびビジネス プロセス フローを含むフロー タイプを特定する  
    (Identify flow types including cloud, desktop, and business process flows)
  - フローのユース ケースと利用可能なフロー テンプレートを説明する  
    (Describe use cases for flows and available flow templates)
  - Power Automate がコネクタ トリガーやアクションをどのように使用するかを説明する  
    (Describe how Power Automate uses connector triggers and actions)
  - ループや switch、do until、apply to each を含む条件を説明する  
    (Describe loops and conditions including switch, do until, and apply to each)
  - 式を説明する  
    (Describe expressions)
  - 承認のユース ケースを説明する  
    (Describe use cases for approvals)
  - Power Automate Desktop、Power Automate モバイル、および Power Automate ポータルを含む Power Automate アプリを説明する  
    (Describe the Power Automate apps, including Power Automate Desktop, Power Automate mobile, and Power Automate portal)
- 基本的なクラウド フローをビルドする  
  (Build a basic cloud flow)
  - インスタントや自動化、もしくはスケジュールされたフロー テンプレートを用いてフローを作成する  
    (Create a flow by using an instant, automated, or scheduled flow template)
  - クラウド フローを修正する  
    (Modify a cloud flow)
  - データ操作を実行するためにフロー制御を使用する  
    (Use flow controls to perform data operations)
  - クラウド フローを実行する  
    (Run a cloud flow)
#### Power Virtual Agents の機能をデモンストレーションする (Demonstrate the capabilities of Power Virtual Agents) (5-10%)
- Microsoft Teams の Power Virtual Agents の機能を説明する  
  (Describe the capabilities of Power Virtual Agents in Microsoft Teams)
  - Microsoft Teams の Power Virtual Agents のユース ケースを説明する  
    (Describe use cases for Power Virtual Agents within Microsoft Teams)
  - トピック、エンティティ、アクションを説明する  
    (Describe topics, entities and actions)
  - メッセージ ノード、質問ノード、条件、トリガー フレーズ、およびオーサリング キャンバスを説明する  
    (Describe message nodes, question nodes, conditions, trigger phrases, and the authoring canvas)
- 基本的なチャットボットをビルド、公開する  
  (Build and publish a basic chatbot)
  - チャットボットを作成する  
    (Create a chatbot)
  - トピックを作成する  
    (Create a topic)
  - アクションを呼び出す  
    (Call an action)
  - チャットボットから Power Automate フローを起動する  
    (Launch a Power Automate flow from a chatbot)
  - チャットボットを公開する  
    (Publish a chatbot)
