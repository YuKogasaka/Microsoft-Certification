注 : 評価される各スキルに続いて記載している箇条書きの一覧は、スキルの評価項目 (どのように評価しているかを示す) となります。関連するトピックが試験内でカバーされている亜愛があります。

注 : ほとんどの問題は、GA (General Availability) となった機能に関するものです。プレビュー機能が一般的に私用されている場合、それらに関する問題が含まれる場合があります。  
(Note : Most questions cover features that are general availability (GA). The exam may contain questions on Preview features if those features are commonly used.)

### 機能グループ
#### Microsoft Dataverse の構成 (Configure Microsoft Dataverse) (20-25%)
- データ モデルを管理する  
(Manage a data model)
  - 標準、アクティビティ、仮想テーブルを作成、変更する  
  (Create or modify standard, activity, or virtual tables)
  - テーブルの所有権を構成する  
  (Configure tables ownership)
  - 新しいテーブルを作成、もしくは既存のテーブルを変更する
  (Create new tables or modify existing tables)
  - 1:N および N:N を含む、実装する関係のタイプを決定する  
  (Determine which type of relationships to implement including 1: N and N: N)
  - カスケード ルールを含むテーブル リレーションシップの動作を構成する  
  (Configure table relationship behaviors including cascading rules)
  - 新しい関係を作成する、もしくは既存の関係を変更する  
  (Create new relationships or modify existing relationships)
  - 新しい列を作成、もしくは既存の列を変更する  
  (Create new columns or modify existing columns)
  - テーブルの代替キーを作成する  
  (Create alternate keys for tables)
  - テーブル プロパティを構成する  
  (Configure table properties)
  - 接続のロールを構成する  
  (Configure connection roles)
  - ビューを作成、構成する  
  (Create and configure views)
- Dataverse を構成する  
(Configure Dataverse)
  - データバース検索を構成する  
  (Configure Dataverse search)
  - 監査を構成する  
  (Configure auditing)
  - インポート、エクスポート、一括削除を含むデータ管理タスクを実行する  
  (Perform data management tasks including import, export, and bulk deletion)
  - 重複データ検出設定を構成する  
  (Configure duplicate detection settings)
- セキュリティ設定を構成する  
(Configure security settings)
  - ビジネス ユニットを作成、管理する  
  (Create and manage business units)
  - セキュリティ ロールを作成、管理する  
  (Create and manage security roles)
  - ユーザーとチームを作成、管理する  
  (Create and manage users and teams)
  - 重複データ検出設定を構成する  
  (Create and manage column security)
  - 階層セキュリティを構成する  
  (Configure hierarchy security)
  - Azure Active Directory グループ チームを構成する  
  (Configure azure active directory group teams)
  - 共有を管理する  
  (Manage sharing)
  - セキュリティの問題を診断する  
  (Diagnose security issues)
#### Microsoft Power Apps を用いたアプリの作成 (Create apps by using Microsoft Power Apps) (15-20%)
- モデル駆動アプリを作成する  
(Create model-driven apps)
  - 複数のフォーム タイプを作成、構成する  
  (Create and configure multiple form types)
  - 特殊なフォーム コンポーネントを利用する  
  (Use specialized form components)
  - ビューを作成、構成する  
  (Create and configure views)
  - Power BI の埋め込みやレポート定義言語 (RDL) の利用を含む、使用するレポート オプションを決定する
  (Determine which reporting options to use including embedding Power BI or using report definition language (RDL))
  - サイト マップを構成し、モデル駆動型アプリを構成する  
  (Configure site maps and compose model-driven apps)
  - モデル駆動型アプリを共有する  
  (Share a model-driven app)
  - カスタム ページを構成する  
  (Configure custom pages)
  - ビジネス ルールを作成、構成する 
  (Create and configure business rules)
  - 最新の指揮を構成する  
  (Configure modern commanding)
- キャンバス アプリのユース ケースを定義する  
(Define use cases for canvas apps)
  - キャンバス アプリを利用するタイミングを特定する  
  (Identify when to use a canvas app)
  - キャンバス アプリの構造について説明する  
  (Describe canvas app structure)
  - フォーム ナビゲーション、数式、変数とコレクション、およびエラー処理について説明する  
  (Describe form navigation, formulas, variables and collections, and error handling)
  - キャンバス アプリで発生するアクションに基づいて Power Automate フローを実行する  
  (Run power automate flows based on actions that occur in a canvas app)
- Microsoft Power Pages を構築する  
(Build Microsoft Power Pages)
  - ページ、フォーム、およびナビゲーションを構成する  
  (Configure pages, forms, and navigation)
  - 高度な Power Page 機能を構成する  
  (Configure advanced power pages features)
  - Web ロールとページ アクセスを含む Web サイト セキュリティを構成する  
  (Configure website security including web roles and page access)
  - テンプレートの利用例を説明する  
  (Describe use cases for templates)
  - 登録オプションを実装する  
  (Implement registration options)
#### プロセス自動化の作成と管理 (Create and manage process automation) (20–25%)
- クラウド フローを作成する  
(Create cloud flows)
  - クラシック ワークフローをいつ利用するかを含む、フローの種類、ユース ケース、フロー コンポーネントについて説明する  
  (Describe types of flows, use cases, and flow components including when to use a classic workflow)
  - 一般的なコネクタ、トリガー、アクションについて説明する  
  (Describe common connectors, triggers, and actions)
  - 分岐、ループ、条件、エラー処理、変数を含むロジック コントロールを実装する  
  (Implement logic controls including branches, loops, conditions, error handling, and variables)
  - 動的なコンテンツと式を実装する  
  (Implement dynamic content and expressions)
  - フローをテスト、トラブルシュートする  
  (Test and troubleshoot flows)
- ビジネス プロセス フローの作成、管理する  
(Create and manage business process flows)
  - 業務プロセス フローを構成する  
  (Configure a business process flow)
  - ビジネス プロセス フローにステージ、ワークフロー、およびアクション ステップを追加する  
  (Add stages, workflows, and action steps to a business process flow)
  - 業務プロセス フローの業務プロセス フロー テーブルを管理する  
  (Manage the business process flow table for a business process flow)
  - 高度なロジックを構成する  
  (Configure advanced logic)
- クラシック ワークフローを作成する  
(Create classic workflows)
  - ワークフローを構成する  
  (Configure a workflow)
  - ワークフローをトラブルシュートする  
  (Troubleshoot workflows)
  - ワークフロー ログを管理する  
  (Manage workflow logs)
#### Microsoft Power Virtual Agents の説明 (Describe Microsoft Power Virtual Agents) (10-15%)
- Power Virtual Agents のユース ケースを特定する  
(Identify use cases for Power Virtual Agents)
  - チャットボット環境を特定する  
  (Identify chatbot environments)
  - チャットボットのスキルと AI 機能について説明する  
  (Describe chatbot skills and AI capabilities)
  - セキュリティのオプションを特定する  
  (Identify options for security)
- Power Virtual Agents のコンポーネントを特定する  
(Identify components of Power Virtual Agents)
  - トピックを説明する
  (Describe topics)
  - エンティティを説明する  
  (Describe entities)
  - フォールバック トピックのユース ケースを特定する  
  (Identify use cases for fallback topics)
#### データフローと Power BI を用いたデータのインポートとビジュアル化 (Import and visualize data by using dataflows and Power BI) (10-15%)
- Power Platform で Power BI を利用する  
(Consume Power BI in Power Platform)
  - モデル駆動型アプリとキャンバス アプリに Power BI タイルを追加する  
  (Add Power BI tiles to model-driven apps and canvas apps)
  - キャンバス アプリを Power BI ダッシュボードに追加する  
  (Add canvas apps to a Power BI dashboard)
  - Power BI アラートから Power Automate フローをトリガーする  
  (Trigger Power Automate flows from Power BI alerts)
  - Power ページで Power BI を利用する  
  (Use Power BI in Power Pages)
- データフローを作成して利用する  
(Create and use dataflows)
  - Power Query を用いて Dataverse にデータをインポートする  
  (Import data to Dataverse using Power Query)
  - データフローの実行をスケジュールする  
  (Schedule dataflow runs)
  - Power Fx について説明する  
  (Describe Power Fx)
#### 環境戦略の定義 (Define an environment strategy) (10-15%)
- アプリケーション ライフサイクル管理 (ALM) に参加する  
(Participate in Application Lifecycle Management (ALM))
  - App Checker と Solution Checker の利用例を説明する  
  (Describe use cases for App Checker and Solution Checker)
  - ソリューションを作成、管理する  
  (Create and manage solutions)
  - マネージド ソリューションとアンマネージド ソリューションの違いについて説明する  
  (Describe the difference between managed and unmanaged solutions)
- 他のサービスとの相互運用性について説明する  
(Describe interoperability with other services)
  - アプリを Microsoft Teams に追加する  
  (Add apps to Microsoft Teams)
  - Teams チャネルを作成する  
  (Create Teams channels)
  - Power Automate Gateway を利用するタイミングを特定する  
  (Identify when to use Power Automate Gateway)
