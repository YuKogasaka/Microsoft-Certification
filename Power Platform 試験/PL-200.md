### 試験の対象者
この試験の受験者は、Discovery を実行し、要件を把握し、対象分野の専門家や利害関係者を関与させ、要件を翻訳、Power Platform ソリューションとアプリを構成します。 これらは、アプリケーションの機能強化、カスタム ユーザー エクスペリエンス、システム統合、データ変換、カスタムプロセスの自動化、およびカスタムの視覚化を作成します。

受験者は、ソリューション アーキテクトによって提供され、それらと協力して設計を実装し、ユーザー エクスペリエンス デザイナーによって確立された標準、ブランディング、アーティファクトを実装します。 サードパーティのアプリケーションやサービスとのシームレスな統合を提供する統合を設計します。

受験者は、品質保証チームのメンバーと積極的に協力して、ソリューションが機能要件と非機能要件を満たしていることを確認します。 パッケージ化と展開のためのアーティファクトを特定して生成し、DevOps エンジニアに提供し、Power Platform 管理者に運用と保守のトレーニングを提供します。

### スキルの評価
注：評価される各スキルの下に記載しているリストは、スキルの評価項目です。このリストは、決定的でも網羅的でもありません。

注：試験内容について、ほとんどの場合、プレビュー機能はカバーされず、一部の機能は GA (General Availability) となった際に試験に追加されます。

#### Microsoft Dataverse の構成 (20-25%)
- データ モデルを管理する
  - 標準、アクティビティ、または仮想をを含むテーブル タイプを割り当てる
  - テーブル オーナーシップを構成する
  - 新しいテーブルを作成する、もしくは既存のテーブルを変更する
  - 1:N と N:N を含む実装するリレーションシップのタイプを決定する
  - カスケード ルールを含むテーブル リレーションシップの動作を構成する
  - 新しいリレーションシップを作成する、もしくは既存のリレーションシップを修正する
  - 新しい列を作成する、もしくは既存の列を修正する
  - テーブルの代替キーを作成する
  - テーブル プロパティを構成する
  - 接続ロールを構成する
  - ビューを作成、構成する
- ロジックを作成、管理する
  - ビジネス ルールの要件を定義する
  - ビジネス ルール ロジックを定義、実装する
  - ビジネス ルールのスコープを定義する
  - ビジネス ルールを構成、テストする
  - 同期クラシック ワークフローを構成する
- Dataverse を構成する
  - Relevance Search を構成する
  - 監査を構成する
  - インポート、エクスポートおよび一括検出を含むデータ管理タスクを実行する
  - 重複検出設定を構成する
  - プライバシー設定を構成する
- セキュリティ設定を構成する
  - ビジネスユニットを作成、管理する
  - セキュリティ ロールを作成、管理する
  - ユーザーとチームを作成、管理する
  - 列セキュリティを作成、管理する
  - 階層セキュリティを構成する
  - Azure AD グループ Teams を構成する
#### Power Apps を用いた アプリの作成 (15-20%)
- モデル駆動アプリを作成する
  - フォームを作成、構成する
  - ビューを作成、構成する
  - チャートを作成、構成する
  - ダッシュボードを作成、構成する
  - サイト マップを構成する
  - テーブル、フォーム、ビュー、ビジネス プロセス フロー、ダッシュボード、グラフを含む、アプリに適用可能なアセットを選択する
  - モデル駆動アプリを共有する
  - テーマを用いて組織のブランドを適用する
- キャンバス アプリを作成する
  - キャンバス アプリを作成する
  - アプリのデータソースとして Datverse を構成する
  - キャンバス アプリ スクリーンを作成する
  - フォーム ナビゲーション、数式、変数とコレクション、およびエラー ハンドリングを実装する
  - 再利用可能なコンポーネントとコンポーネントライブラリを構築する
  - アプリのオフライン機能を構成する
  - キャンバス アプリで発生するアクションに基づいて Power Automate フローを実行する
  - App Checker の結果を解釈し、アクセシビリティの問題を含む特定された問題を解決する
  - アプリを管理する
  - アプリ分析データを監視する
- ポータル アプリを作成する
  - ポータル アプリを作成する
  - Dataverse データを公開する
  - ポータルの web ページ、フォーム、ナビゲーションを構成する
  - web ロールやページ アクセスを含む ポータル セキュリティを構成する
  - ポータルの詳細とアクションを構成する
  - ポータルの認証を構成する
#### Power Automate の作成と管理 (15-20%)
- フローを作成する
  - どのタイミングでクラシックワークフローを使用するかどうかを含む、フローのタイプとフロー コンポーネントを推奨する
  - Dataverse コネクターを用いてフローをトリガーする
  - Dataverse コネクターを用いてアクションを実行する
  - ブランチ、ループ、条件、エラー ハンドリングおよび変数を含む、ロジック制御を実装する
  - 動的コンテンツと式を実装する
  - フローをテスト、トラブルシュートする
- ビジネス プロセス フローを作成、管理する
  - ビジネス プロセス フローを構成する
  - ビジネス プロセス フローにビジネス ルール、ワークフロー、フロー ステップ、アクション ステップを追加する
  - ステージとステップを定義する
  - 並列ブランチを構成する
  - ビジネス プロセス フローのビジネス プロセス フロー テーブルを管理する
- デスクトップ フローをビルドする
  - デスクトップ フローのタイプを説明する
  - 有人デスクトップ フローと無人デスクトップ フローの区別を含む、デスクトップ フローのユースケースを識別する
  - Power Automate Desktop を用いて Web とユーザー インターフェースの自動化をビルドする
  - Power Automate Desktop フローに変数、ループ、および条件を実装する
  - Cloud フローからデスクトップ フローをトリガーする
  - 自動実行を監視する
  - Process Advisor を用いて処理を分析する
#### Power Virtual Agent チャットボットの実装 (10-15%)
- チャットボットを作成する
  - スタンドアロン チャットボットを作成する
  - Teams や他のチャネルにスタンドアロン チャットボットを追加する
  - Microsoft Teams チャネル内にチャットボットを作成する
  - チャットボットを公開する
  - チャットボットを共有する
  - ボットのパフォーマンス、使用状況、トピックの使用状況を監視、診断する
  - チャットボットに対してエンドユーザーを認証する
- トピックを構成する
  - トピック会話トリガーを定義する
  - 質問、メッセージ、条件を作成する
  - web ページからトピックを抽出する
  - 挨拶、エスカレーション、エラー メッセージ、およびステータスを実装する
  - アクションを実行するために Power Automate フローを呼び出す
  - フォールバック トピックを構成する
- エンティティを構成する
  - カスタム エンティティを作成する
  - 会話にエンティティを実装する
  - データを格納する変数を実装する
#### Power Apps と他のアプリやサービスとの統合 (10-15%)
- Power BI と Power Platform コンポーネントを統合する
  - Power BI のヴィジュアル、レポートおよびダッシュボードを作成する
  - モデル駆動型アプリとキャンバス アプリに PowerBI タイルを追加する
  - Power BI ダッシュボードにキャンバス アプリを追加する
  - Power BI アラートから Power Automate フローをトリガーする
- AI Builder を実装する
  - 使用する AI Builder モデル タイプを決定する
  - AI Builder モデルを作成する
  - モデルで使用するためのソース データを準備する
  - モデルのトレーニング、テスト、公開する
  - Power Apps を用いてモデルを使用する
  - Power Automate を用いてモデルを使用する
- テンプレートを構成、利用する
  - Microsoft Excel テンプレートを構成、利用する
  - Microsoft Word および Microsoft Excel テンプレートを構成、利用する
  - 電子メール テンプレートを構成、利用する
#### ソリューションの管理 (15-20%)
- 開発環境にソリューションを作成する
  - ソリューション資産を含むソリューションを作成する
  - 発行者を作成する
  - ソリューションに資産を追加する
  - ソリューション対応コンポーネントを構築する
  - ソリューション コンポーネントの依存関係を管理する
- 環境間のソリューションを転送する
  - 接続参照を解決する
  - 環境変数を設定する
  - ソリューションをエクスポートする
  - ソリューションをインポートする
  - ソリューションをアップデートする
  - 管理プロパティを構成する
  - Solution Checker を実行し、結果を解釈する
- ソリューションをローカライズする
  - 通貨を構成する
  - 言語パックを有効化する
  - 翻訳をエクスポート、インポートする
